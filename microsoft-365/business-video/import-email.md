# <a name="import-email-to-microsoft-365-for-business"></a>Importer le courrier électronique vers Microsoft 365 pour les entreprises 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/40f7df36-9e24-44e5-8791-e9ed0dd8fd21?autoplay=false]

Une fois que vous vous êtes inscrit à Microsoft 365, il se peut que vous ayez encore des courriers vers un ancien compte ; par exemple, si vous avez utilisé un compte Gmail ou Outlook pour le travail. Vous pouvez continuer à utiliser à la fois les nouveaux comptes et les anciens comptes, ou vous pouvez importer les données de l’ancien compte vers le nouveau.

## <a name="try-it"></a>Essayez !

### <a name="step-1-export-your-old-email"></a>Étape 1 : exporter votre ancien courrier électronique.

1. Dans Outlook, sélectionnez  **fichier**, **ouvrir et exporter**, puis **Importer/exporter**.
2. Sélectionnez  **Exporter vers un fichier** , puis cliquez sur  **suivant**.
3. Sélectionnez  **fichier de données Outlook** , puis  **suivant**.
4. Sélectionnez votre ancien compte de courrier.
5. Vérifiez que vous incluez des sous-dossiers. Ensuite, sélectionnez  **suivant**.
6. Sélectionnez  **Parcourir**, puis accédez à un emplacement facile à mémoriser, comme votre bureau. Sélectionnez  **OK**, puis **Terminer**.
7. Laissez les champs du mot de passe vides, puis choisissez **OK**.

    Votre ancien compte de courrier a été exporté.

### <a name="step-2-import-your-old-email-into-your-new-email-account"></a>Étape 2 : importer votre ancien courrier électronique dans votre nouveau compte de messagerie

1. Dans Outlook, sélectionnez  **fichier**, **ouvrir**,  **Exporter**, puis **Importer/exporter**.
2. Sélectionnez  **Importer à partir d’un autre programme ou fichier** , puis cliquez sur  **suivant**.
3. Sélectionnez  **fichier de données Outlook** , puis  **suivant**.
4. Sélectionnez  **Parcourir**, puis accédez au fichier que vous venez d’exporter. Sélectionnez-le, puis cliquez sur  **ouvrir**, puis sur **suivant**.
5. Vérifiez que vous incluez des sous-dossiers et que vous importez des éléments dans le même dossier dans votre nouveau compte de messagerie. Cliquez sur **Terminer**.
