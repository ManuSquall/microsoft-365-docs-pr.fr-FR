# <a name="install-outlook-and-office-apps-on-ios-devices"></a>Installer des applications Outlook et Office sur des appareils iOS

Une fois que vous avez&#39;la configuration de Microsoft 365 Business Premium, vous pouvez installer des applications Office sur votre iPhone. Après avoir installé les applications, vous&#39;être invité à créer un code confidentiel pour sécuriser vos données Office.

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWee2n?autoplay=false]

## <a name="try-it"></a>Essayez !

1. Sur votre iPhone, ouvrez le magasin d’applications.
2. Sélectionnez  **recherche** , entrez  **Outlook** dans la barre de recherche, puis sélectionnez-le.
3. Choisissez **obtenir** l'   >   **installation**.
4. Entrez votre mot de passe d’ID Apple, sélectionnez **connexion**  >   **ouverte**.
5. Entrez votre adresse de messagerie professionnelle.
6. Sélectionnez  **Ajouter un compte** , entrez votre mot de passe, puis cliquez sur  **se connecter**.
7. (Facultatif), sélectionnez * * faisons-le * * pour ajouter d’autres comptes ou sélectionnez-le  **ultérieurement**  pour ignorer cette étape.
8. Sélectionnez  **OK** pour redémarrer l’application afin que Microsoft 365 entreprise puisse protéger vos données Outlook.
9. Ouvrez de nouveau Outlook et un message de confirmation s’affiche. Sélectionnez  **OK**.
10. Créez un code confidentiel pour Outlook et d’autres applications Office, puis confirmez-le. À partir de maintenant, ce code confidentiel sera utilisé pour sécuriser toutes les données de l’application Office.
11. Sélectionnez  **activer**  pour activer les notifications par courrier électronique.
12. Sélectionnez  **autoriser** pour activer les notifications de type transmission.

Vous pouvez désormais utiliser Outlook sur votre iPhone pour vérifier votre courrier, votre calendrier et vos contacts.

Vous pouvez installer les autres applications Office de la même manière et les données de ces applications seront protégées par Microsoft 365 Business Premium.
