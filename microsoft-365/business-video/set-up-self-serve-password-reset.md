# <a name="set-up-self-service-password-reset"></a>Configurer la réinitialisation du mot de passe en libre-service

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3AY8S?autoplay=false]

Pour permettre à vos utilisateurs de réinitialiser leur mot de passe sans avoir à vous contacter à chaque fois, vous pouvez configurer la réinitialisation du mot de passe en libre-service dans Microsoft 365 Business Premium.

## <a name="try-it"></a>Essayez !

1. Dans le centre d’administration 365 de Microsoft, dans le volet de navigation de gauche, sélectionnez **paramètres** , puis **sécurité & confidentialité**.
1. Sous **permettre à vos personnes de réinitialiser leur mot de passe**, sélectionnez **Centre d’administration Azure ad**.
1. Dans le volet de navigation de gauche, sélectionnez **utilisateurs**, puis, dans la page **utilisateurs-tous les utilisateurs** , sélectionnez **Réinitialiser le mot de passe**.
1. Sélectionnez **tous** pour activer la réinitialisation du mot de passe libre-service, puis sélectionnez **Enregistrer**.

La prochaine fois qu’un utilisateur se connecte à son compte, il est invité à entrer son ID utilisateur et son mot de passe. Sélectionnez **suivant**, puis indiquez s’il faut s’authentifier avec leur téléphone, leur messagerie ou les deux. Ils entrent le code qu’ils reçoivent, sélectionnez **vérifier**, puis **Terminer**. Une fois qu’ils sont effectués, ils peuvent réinitialiser leur mot de passe.
