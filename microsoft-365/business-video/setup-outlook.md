# <a name="set-up-outlook-for-microsoft-365-for-business-email"></a>Configurer Outlook pour Microsoft 365 pour le courrier électronique professionnel 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/9fe86884-8a83-42cc-bca9-61a12e6dad31?autoplay=false]

## <a name="try-it"></a>Essayez !

Après avoir installé les applications Office, vous&#39;souhaitez configurer Outlook de sorte qu’il commence à utiliser la messagerie, le calendrier et les contacts. Ici&#39;s.

1. Ouvrez le menu Démarrer. Recherchez Outlook, puis sélectionnez-le.
2. Entrez votre adresse de messagerie Microsoft 365, puis sélectionnez  **connecter**.
3. Entrez les adresses de messagerie supplémentaires que vous souhaitez utiliser, telles que votre adresse de messagerie précédente ou personnelle. Sélectionnez  **suivant**.
4. Si vous y êtes invité, entrez un mot de passe, puis sélectionnez  **connexion**.
5. Une fois tous vos comptes ajoutés, choisissez si vous souhaitez configurer Outlook Mobile ou patienter jusqu’à la fin de l’installation.
6. Sélectionnez  **Terminer**. Outlook peut prendre plusieurs minutes pour télécharger votre courrier électronique et d’autres données.

Vous pouvez désormais afficher le courrier électronique pour les comptes que vous avez ajoutés. Vous pouvez également afficher votre calendrier, vos contacts et vos tâches.
