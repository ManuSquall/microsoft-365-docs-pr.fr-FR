# <a name="set-up-multi-factor-authentication-on-your-phone"></a>Configurer l’authentification multifacteur sur votre téléphone

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2MmQR?autoplay=false]

L’authentification multifacteur offre davantage de sécurité pour votre entreprise. Procédez comme suit pour le configurer.

## <a name="try-it"></a>Essayez !

1. Lorsque vous vous connectez à [https://office.com](https://office.com) , vous verrez l’invite **more information required** . Cliquez sur **Suivant**.
1. Sous **étape 1**, choisissez **application mobile** dans la liste déroulante **téléphone d’authentification** .
1. Sous Comment voulez-vous utiliser l’application mobile ?, sélectionnez la méthode que vous souhaitez utiliser pour vous connecter :
    - Sélectionnez **recevoir les notifications de vérification** pour s’authentifier directement à partir de votre application mobile, qui peut inclure l’authentification par empreinte digitale.
    - Sélectionnez **utiliser le code de vérification** pour entrer un nouveau code de vérification chaque fois que vous vous authentifiez.
1. Sélectionnez **Configurer**.
1. Laissez la fenêtre **configurer l’application mobile** ouverte sur votre ordinateur.
1. Sur votre appareil mobile, accédez au magasin d’applications, recherchez « Microsoft Authenticator », sélectionnez-le, installez-le, puis ouvrez-le.
1. Dans l’authentificateur, suivez les invites pour terminer l’Assistant, puis choisissez le signe plus (+) pour ajouter votre compte.
1. Choisissez un **compte professionnel ou scolaire**, choisissez **autoriser** lorsque vous êtes invité à prendre des photos et enregistrer une vidéo, puis suivez les instructions pour numériser le code QR dans la fenêtre ouverte de votre ordinateur.
1. Une fois que le compte a été ajouté, sélectionnez **obtenu**.
1. Sur votre ordinateur, cliquez sur **suivant**, puis de nouveau sur **suivant** .
1. Notez le code qui s’affiche sur votre appareil mobile, entrez le code sur votre ordinateur dans le champ à l' **étape 2**, puis cliquez sur **vérifier**.
1. Dans les zones de l' **étape 3**, entrez un numéro de téléphone de sauvegarde, comme votre numéro de bureau, choisissez **suivant**, puis sélectionnez **OK**.
