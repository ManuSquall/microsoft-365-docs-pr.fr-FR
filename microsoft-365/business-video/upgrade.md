# <a name="upgrade-from-windows-10-home-to-windows-10-pro"></a>Mise à niveau de Windows 10 Home vers Windows 10 professionnel

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j?autoplay=false]

## <a name="try-it"></a>Essayez !

Microsoft 365 entreprise requiert Windows 10 professionnel pour sécuriser vos données. Si vous disposez d’un ordinateur exécutant Windows 10 Home, procédez comme suit pour effectuer une mise à niveau vers Windows 10 professionnel.

1. À partir de votre bureau, choisissez le menu  **Démarrer**  , recherchez  **Microsoft Store** , puis sélectionnez-le.
2. Dans le Microsoft Store, cliquez sur dans le volet de recherche, tapez  **Windows 10 professionnel** , puis choisissez  **mettre à niveau vers Windows 10 professionnel**.
3. Sous  **mettre à niveau vers Windows 10 professionnel** , sélectionnez  **en savoir plus** , puis choisissez  **acheter**.
4. Dans la zone  **sécurité Windows**  , entrez votre code confidentiel pour vérifier votre identité.
5. Consultez la page paiement, lisez les conditions de vente, modifiez votre mode de paiement si vous le souhaitez, puis choisissez  **acheter**.
6. Sous  **mettre à niveau vers Windows 10 professionnel** , choisissez  **installer**.
7. Enregistrez tous les fichiers si nécessaire, puis cliquez sur  **Oui, laissez&#39;s**  commencer à installer Windows.
