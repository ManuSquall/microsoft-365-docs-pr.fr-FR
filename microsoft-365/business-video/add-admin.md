# <a name="add-an-admin"></a>Ajouter un administrateur

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FOfO?autoplay=false]

Lorsque vous vous inscrivez à Microsoft 365 entreprise, vous devenez automatiquement administrateur général. Pour faciliter la gestion de l’entreprise, vous pouvez également créer d’autres administrateurs de personnes. 

## <a name="try-it"></a>Essayez !

### <a name="add-an-existing-employee-as-an-admin"></a>Ajouter un employé existant en tant qu’administrateur

1. Lorsque vous vous inscrivez à Microsoft 365 entreprise, vous devenez automatiquement administrateur général. Pour faciliter la gestion de l’entreprise, vous pouvez également créer d’autres administrateurs de personnes. 
1. Dans le centre d’administration 365 de Microsoft, sélectionnez **utilisateurs**, puis **utilisateurs actifs**.
1. Sélectionnez l’utilisateur pour lequel vous souhaitez créer un administrateur, puis sélectionnez **gérer les rôles**.
1. Dans le volet **gérer les rôles** , désactivez la case à cocher rôle actuel de l’utilisateur, puis sélectionnez le nouveau rôle à attribuer à l’utilisateur.
1. Sélectionnez **enregistrer les modifications**, puis fermez le volet de confirmation **mise à jour des rôles d’administrateur** .

### <a name="add-someone-outside-the-company-as-an-admin"></a>Ajouter une personne en dehors de l’entreprise en tant qu’administrateur

S’il existe une personne externe qui aidera à gérer votre entreprise, vous pouvez l’ajouter en tant qu’administrateur général.

1. Dans le centre d’administration 365 de Microsoft, sélectionnez **utilisateurs**, **utilisateurs actifs**, puis **Ajoutez un utilisateur**.
1. Dans la page **configurer les notions de base** , entrez le nom de l’utilisateur, le nom d’affichage et le nom d’utilisateur.
1. Activez la case à cocher **Envoyer le mot de passe dans un message électronique** à la fin de l’opération pour que le mot de passe de l’utilisateur lui soit envoyé. Tapez son adresse de messagerie. Séparez les adresses par des points-virgules. Sélectionnez **Suivant**.
1. Dans la liste déroulante **Sélectionner un emplacement** , choisissez l’emplacement de l’utilisateur.
1. Activez la case à cocher **créer un utilisateur sans licence de produit** . Vous n’avez pas besoin d’attribuer une licence à cet utilisateur car il gère uniquement les services Microsoft 365, et ne les utilise pas. Sélectionnez **Suivant**.
1. Développez la section **rôles** . Désactivez la case à cocher en regard du rôle actuel de l’utilisateur, puis choisissez **administrateur général**. Sélectionnez **Suivant**.
1. Vérifiez vos paramètres, puis sélectionnez **terminer l’ajout**.
1. Sélectionnez **Fermer**.

Votre nouvel administrateur général peut désormais vous aider à gérer vos services Microsoft 365.
