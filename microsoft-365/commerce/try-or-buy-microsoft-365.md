# <a name="try-or-buy-a-microsoft-365-for-business-subscription"></a>Essayer ou acheter un abonnement Microsoft 365 pour les entreprises

::: moniker range="o365-21vianet"

> [!NOTE]
> Le centre d’administration change. Si votre expérience ne correspond pas aux informations présentées ici, voir [À propos du nouveau centre d’administration Microsoft 365](../admin/microsoft-365-admin-center-preview.md?view=o365-21vianet).

::: moniker-end

Microsoft 365 pour les entreprises est un service d’abonnement qui vous permet d’exécuter votre organisation dans le Cloud pendant que Microsoft s’occupe de l’informatique pour vous. Microsoft gère les appareils, protège contre les menaces du monde réel et fournit à votre organisation les derniers logiciels d’entreprise. Vous pouvez vous inscrire pour obtenir un abonnement d’évaluation gratuit pour Microsoft 365 Business standard, Microsoft 365 Business Premium ou Microsoft 365 apps pour les entreprises et essayer pendant 30 jours.

> [!NOTE]
> Vous devez utiliser une carte de crédit lorsque vous vous inscrivez à une version d’évaluation gratuite. À la fin de votre période d’évaluation gratuite, votre abonnement à la version d’évaluation est automatiquement converti en abonnement payant. Votre carte de crédit n’est facturée qu’au terme de la période d’évaluation.

## <a name="before-you-begin"></a>Avant de commencer

Vous n’avez pas besoin d’un compte Microsoft existant pour vous inscrire à une version d’évaluation gratuite. Pour toutes les autres procédures décrites dans cet article, vous devez être un administrateur général ou un administrateur de facturation pour votre organisation. Pour plus d’informations, consultez la rubrique [à propos des rôles d’administrateur](../admin/add-users/about-admin-roles.md).

## <a name="try-a-free-trial-subscription"></a>Essayer un abonnement gratuit à la version d’évaluation

Vous êtes un nouveau client, et vous n’avez pas encore de compte auprès de Microsoft ? Suivez ces étapes pour créer un compte et vous inscrire pour obtenir un abonnement d’évaluation gratuit de Microsoft 365 Business standard, Microsoft 365 Business Premium ou Microsoft 365 apps pour les entreprises.

1. Accédez au <a href="https://www.aka.ms/office365signup" target="_blank">site produits Microsoft 365</a>.
2. Sélectionnez le plan pour lequel vous souhaitez vous inscrire, tel que **Microsoft 365 Business standard**, faites défiler la page vers le bas, puis sélectionnez **essayer gratuitement pendant 1 mois**.
3. Sur la page suivante, suivez les étapes pour configurer votre compte.
4. Le processus d'inscription peut prendre plusieurs minutes. Une fois \' l’opération terminée, vous êtes prêt à démarrer l’Assistant Installation de votre abonnement. Pour plus d’informations sur la configuration de votre abonnement, voir [étapes suivantes](#next-steps).

## <a name="buy-a-subscription-from-your-free-trial"></a>Acheter un abonnement depuis votre version d’essai

À la fin de votre période d’évaluation gratuite, votre abonnement à la version d’évaluation est automatiquement converti en abonnement payant. L’abonnement payant est défini par défaut sur le plan dont vous disposez actuellement. Vous pouvez acheter une autre offre en suivant les étapes de la procédure [acheter un autre abonnement](#buy-a-different-subscription).

Si vous souhaitez acheter votre abonnement avant la version d’évaluation, procédez comme suit :

1. Dans le centre d’administration Microsoft 365, accédez à la page **facturation**   \>  <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">de vos produits</a>   .
2. Sur la page **vos produits**   , recherchez l’abonnement que vous souhaitez acheter.
3. Dans la **Licenses**   section licences, sélectionnez **abonnement achat**.
4. Choisissez un engagement mensuel ou annuel pour votre abonnement, puis sélectionnez **valider**.
5. Sur la page suivante, vérifiez l’abonnement, puis sélectionnez **valider**.
6. Sur la page suivante, vérifiez l’adresse **vendue**   , les informations à **facturer**   et les **éléments dans cet ordre**. Si vous avez besoin d’effectuer des modifications, sélectionnez **modifier**   en regard de la section applicable.
7. Lorsque vous avez \' terminé, sélectionnez **accepter l’accord & passer une commande**.

## <a name="cancel-your-free-trial-subscription"></a>Annuler votre abonnement à la version d’évaluation gratuite

Si vous décidez d’annuler votre abonnement d’évaluation avant la fin de la période d’essai gratuite, accédez au centre d’administration de Microsoft 365 et désactivez la [facturation périodique](subscriptions/renew-your-subscription.md#turn-recurring-billing-off-or-on). La version d’évaluation expire automatiquement à la fin de votre mois et votre carte de crédit n' \' est pas facturée.

## <a name="try-a-different-subscription"></a>Essayer un autre abonnement

Si vous disposez déjà d’un abonnement Microsoft 365 pour les entreprises, vous pouvez utiliser le centre d’administration Microsoft 365 pour essayer un autre abonnement.

Lorsque vous ajoutez un abonnement via le centre d’administration Microsoft 365, le nouvel abonnement est associé à la même organisation (espace de noms de domaine) que votre abonnement existant. Cette association permet de déplacer plus facilement les utilisateurs de votre organisation entre les abonnements, ou de leur attribuer une licence pour les produits supplémentaires dont ils ont besoin.

1. Dans le centre d’administration, accédez à **Billing**la   \> page <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">services d’achat</a>   de facturation.
2. Dans la page **acheter des services**   , vous pouvez voir les offres disponibles pour votre organisation. Choisissez le plan Microsoft 365 que vous souhaitez essayer.
3. Sur la page suivante, sélectionnez **obtenir une version d’évaluation gratuite**. La version d’évaluation vous donne 25 licences utilisateur pour un délai d’un mois.
4. Choisissez de recevoir un texte ou un appel, entrez votre numéro de téléphone, puis choisissez **me texte**   ou **m’appeler**.
5. Entrez le code de vérification, puis sélectionnez **Démarrer votre version d’évaluation gratuite**.
6. Sur la **Check out**   page extraire, sélectionnez **essayer maintenant**.
7. Sur la page **bon de commande**   , sélectionnez **Continuer**.

## <a name="buy-a-different-subscription"></a>Acheter un autre abonnement

Si vous disposez déjà d’un abonnement Microsoft 365 pour les entreprises, vous pouvez passer par le centre d’administration 365 Microsoft pour acheter un autre abonnement pour votre organisation.

Lorsque vous achetez un autre abonnement via le centre d’administration, le nouvel abonnement est associé à la même organisation (espace de noms de domaine) comme abonnement existant. Ainsi, il est plus facile de déplacer les utilisateurs de votre organisation entre les abonnements ou de leur attribuer une licence pour l’abonnement supplémentaire dont ils ont besoin.

1. Dans le centre d’administration, accédez à **Billing**la   \> page <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">services d’achat</a>   de facturation.
2. Dans la page **acheter des services**   , sélectionnez le plan que vous souhaitez acheter, sélectionnez **Détails**, puis **acheter**.
3. Entrez le nombre de licences dont vous avez besoin et indiquez si vous devez payer chaque mois ou pour l’ensemble de l’année. Choisissez si vous souhaiter attribuer automatiquement des licences à tous vos utilisateurs qui n’en ont pas actuellement. Ensuite, sélectionnez **extraire**.
4. Passez en revue les informations de tarification, puis sélectionnez **suivant**.
5. Fournissez vos informations de paiement, puis sélectionnez passer la **commande**   \>  **sur Accueil de l’administration**.

> [!NOTE]
> Vous devez déplacer les utilisateurs de votre abonnement d’évaluation gratuit vers le nouvel abonnement avant la fin de votre période de grâce de 90 jours après l’expiration de votre abonnement d’évaluation. En procédant ainsi, vous conservez vos données, comptes et configuration. Dans le cas contraire, ces informations sont supprimées.
