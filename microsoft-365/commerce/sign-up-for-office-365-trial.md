# <a name="sign-up-for-an-office-365-trial-subscription"></a>Inscription à un abonnement d’évaluation Office 365

Vous pouvez vous inscrire pour obtenir gratuitement un abonnement Office 365 Business. Vous pouvez le tester pendant 30 jours.

## <a name="sign-up-for-an-office-365-business-premium-trial-subscription"></a>Inscription à un abonnement à la version d’évaluation d’Office 365 Business Premium

1. Pour vous inscrire à un abonnement d’évaluation Office 365 Business, accédez au [site produits Office](https://www.aka.ms/office365signup). 
    
2. Sélectionnez le plan pour lequel vous souhaitez vous inscrire, par exemple **Office 365 Business Premium**, faites défiler la page vers le bas, puis sélectionnez **essayer gratuitement pendant 1 mois**.
    
3. Sur la page suivante, suivez les étapes de configuration de votre compte.

4. Le processus d’inscription peut prendre plusieurs minutes. Une fois l’opération terminée, vous trouverez plus d’informations sur la façon de commencer à utiliser votre abonnement.

Pour plus d’informations sur la configuration de votre compte, voir [Configurer Office 365 pour les entreprises](../admin/setup/setup.md).
