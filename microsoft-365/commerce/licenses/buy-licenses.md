# <a name="buy-or-remove-subscription-licenses"></a>Acheter ou supprimer des licences d’abonnement

::: moniker range="o365-21vianet"

> [!NOTE]
> Le centre d’administration change. Si votre expérience ne correspond pas aux informations présentées ici, voir [À propos du nouveau centre d’administration Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/microsoft-365-admin-center-preview?view=o365-21vianet).

::: moniker-end

Vous pouvez acheter des licences supplémentaires ou réduire le nombre de licences pour vos abonnements en procédant comme suit.

## <a name="before-you-begin"></a>Avant de commencer

- Pour effectuer les tâches décrites dans cet article, vous devez être administrateur général ou administrateur de facturation. Pour plus d’informations, consultez [À propos des rôles d’administrateur](../../admin/add-users/about-admin-roles.md).
- Vous pouvez [Ajouter des utilisateurs et attribuer des licences en même temps](../../admin/add-users/add-users.md).

## <a name="buy-or-remove-licenses-for-your-business-subscription"></a>Acheter ou supprimer des licences pour votre abonnement professionnel

::: moniker range="o365-worldwide"

1. Dans le centre d’administration, accédez à la page **Facturation** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Produits</a>.
2. Sur l’onglet **produits** , recherchez l’abonnement pour lequel vous souhaitez acheter ou supprimer des licences. Sélectionnez **autres actions** (trois points), puis sélectionnez **acheter des licences**. [Que faire si le lien Ajouter/Supprimer des licences n'est pas disponible ?](#what-if-i-dont-see-the-addremove-licenses-link)
3. Si vous souhaitez réduire le nombre de licences, en haut du volet acheter des **licences** , sélectionnez **supprimer**des licences.
4. Pour acheter ou supprimer des licences, sous **nouvelle quantité** dans la zone **nombre total de licences** , entrez le nombre total de licences dont vous avez besoin pour cet abonnement. Par exemple, si vous avez 100 licences et que vous souhaitez ajouter cinq autres, entrez 105. Si vous souhaitez en supprimer cinq, entrez 95.
5. Sélectionnez **Enregistrer**.

::: moniker-end

::: moniker range="o365-germany"

1. Dans le Centre d’administration, accédez à la page **Facturation** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Abonnements</a>.
2. Sur la page **abonnements** , sélectionnez l’abonnement auquel vous souhaitez acheter ou supprimer des licences, puis sélectionnez **Ajouter/supprimer des licences**. [Que faire si le lien Ajouter/Supprimer des licences n'est pas disponible ?](#what-if-i-dont-see-the-addremove-licenses-link)
3. Dans la zone nombre **total de licences** , entrez le nombre total de licences souhaitées pour cet abonnement, puis sélectionnez **Envoyer** \> **Close**. Par exemple, si vous avez 100 licences et que vous souhaitez ajouter cinq autres, entrez 105. Si vous souhaitez supprimer cinq licences, entrez 95.

::: moniker-end

::: moniker range="o365-21vianet"

1. Dans le Centre d’administration, accédez à la page **Facturation** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Abonnements</a>.
2. Sur la page **abonnements** , sélectionnez l’abonnement auquel vous souhaitez acheter ou supprimer des licences, puis sélectionnez **Ajouter/supprimer des licences**. [Que faire si le lien Ajouter/Supprimer des licences n'est pas disponible ?](#what-if-i-dont-see-the-addremove-licenses-link)
3. Dans la zone nombre **total de licences** , entrez le nombre total de licences que vous souhaitez pour cet abonnement, puis sélectionnez **Envoyer** \> **Close**. Par exemple, si vous avez 100 licences et que vous devez en ajouter 5, entrez 105. Si vous souhaitez en supprimer 5, entrez 95.

::: moniker-end

> [!NOTE]
> Vous ne pouvez pas réduire le nombre de licences pour votre abonnement si toutes les licences sont actuellement attribuées à des utilisateurs. Pour réduire le nombre de licences, [désaffectez d’abord une ou plusieurs licences aux utilisateurs](../../admin/manage/remove-licenses-from-users.md), puis supprimez les licences de l’abonnement.

|Reason  |Description  |Solution  |
|---------|---------|---------|
|Une vérification de solvabilité est en attente. |Si une vérification de solvabilité est en attente, le message « vérification de solvabilité en attente » s’affiche. Vous ne pouvez pas acheter de licences tant que la vérification de solvabilité n’est pas terminée.  | Vérifiez plus tard pour voir si la vérification du crédit est terminée. Les vérifications de la solvabilité nécessitent en général deux jours ouvrés.<br/>Une fois la vérification du crédit terminée, vous devez voir le lien **Ajouter/supprimer des licences** . |
|Vous avez activé l’abonnement à l’aide d’une clé de produit.| Si l’abonnement a été acheté et activé à l’aide d’une clé de produit de 25 caractères, le texte « prépayé » s’affiche.  |Voir [Ajouter des licences à un abonnement payé pour l’utilisation d’une clé de produit](add-licenses-using-product-key.md). |
|Vous avez acheté votre abonnement auprès d’un partenaire. | Si l’abonnement a été acheté auprès d’un partenaire, le lien Centre de gestion des licences en volume (VLSC) s’affiche. | Voir [Ajouter des licences à un abonnement acheté via le centre de gestion des licences en volume](add-licenses-bought-through-vlsc.md). |
|Vous avez acheté votre abonnement auprès d’un revendeur.|| Si l’abonnement a été acheté via un partenaire de fournisseur de solutions Cloud, contactez votre partenaire CSP pour acheter des licences supplémentaires.        |
|Vous disposez d’un abonnement d’évaluation. |Une version d’évaluation de Microsoft 365 affiche le texte « version d’évaluation ». | Commencez par acheter votre abonnement à la version d’évaluation, puis ajoutez des licences supplémentaires. Consultez [la rubrique acheter un abonnement à Microsoft 365 pour les entreprises à partir de votre version d’évaluation gratuite](../buy-a-subscription-from-your-free-trial.md).|

## <a name="what-you-need-to-know-about-buying-licenses-for-your-business-subscription"></a>Ce que vous devez savoir sur l’achat de licences pour votre abonnement professionnel

### <a name="license-availability"></a>Disponibilité des licences

- **Si vous disposez d’un profil de facturation**: la carte de crédit associée à votre profil de facturation est facturée dès que vous achetez davantage de licences pour un abonnement. Les licences sont immédiatement disponibles pour les utilisateurs. [Qu’est-ce qu’un profil de facturation ?](../billing-and-payments/manage-billing-profiles.md)
- **Si vous n’avez pas de profil de facturation**: Si vous payez votre abonnement par carte bancaire, ou compte bancaire, toutes les nouvelles licences que vous achetez sont immédiatement disponibles après avoir reçu une confirmation de commande. Si vous payez par facture, il est possible que vous deviez attendre la vérification de votre solvabilité avant que les nouvelles licences ne soient disponibles et utilisables.
  > [!NOTE]
  > Le règlement par compte bancaire n’est pas disponible dans certains pays ou régions.
- **Si vous avez prépayé votre abonnement avec une clé de produit**: vous pouvez ajouter des licences supplémentaires en ajoutant une carte bancaire ou un compte bancaire pour traiter les frais supplémentaires des nouvelles licences. Une fois que vous avez acheté les nouvelles licences, nous ajoutons un deuxième abonnement avec le nombre de nouvelles licences que vous venez d’ajouter. Par exemple, si vous avez un abonnement prépayé avec cinq licences, puis acheté 10 licences supplémentaires, vous voyez deux abonnements mentionnés : un avec les cinq licences prépayées et un autre avec les 10 nouvelles licences.

### <a name="changing-your-payment-method"></a>Modification de votre mode de paiement

- **Si vous disposez d’un profil de facturation**: la carte de crédit associée à votre profil de facturation est facturée dès que vous achetez davantage de licences pour un abonnement. [Qu’est-ce qu’un profil de facturation ?](../billing-and-payments/manage-billing-profiles.md)
- **Si vous n’avez pas de profil de facturation :** Si vous payez par carte de crédit, carte bancaire ou compte bancaire, les frais d’achat de nouvelles licences apparaissent sur votre mode de paiement dans les deux jours.

### <a name="billing-statements"></a>Relevés de facturation

- Les licences ajoutées au milieu de votre période de facturation apparaissent sur votre prochaine facture. Si vous payez chaque année, vous êtes facturé dans un délai d’un mois pour ces modifications.
- Sur votre relevé de facturation suivant, les frais précédents pour le nombre d’origine de licences sont déduits. Nous ajoutons un tarif procertain pour la période avec le nombre initial de licences et ajoutons des frais pour le nouveau nombre de licences. Le nombre actuel de licences est également facturé pour le reste de votre période de facturation.

## <a name="next-steps"></a>Étapes suivantes

Si vous avez acheté plus de licences pour votre abonnement, la tâche suivante consiste [à attribuer ces licences aux utilisateurs de votre organisation](../../admin/manage/assign-licenses-to-users.md).

Si vous avez réduit le nombre de licences pour votre abonnement car quelqu’un a quitté votre organisation, vous pouvez supprimer le compte de cet utilisateur. Pour en savoir plus, consultez [la rubrique supprimer un ancien employé](../../admin/add-users/remove-former-employee.md).
