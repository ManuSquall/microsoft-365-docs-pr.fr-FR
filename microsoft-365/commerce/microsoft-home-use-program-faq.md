# <a name="microsoft-home-use-program-frequently-asked-questions-faq"></a>Forum aux questions sur le programme d’utilisation domestique de Microsoft (FAQ)

Les clients de licence en volume utilisent le programme d’utilisation domestique de Microsoft (HUP) sur le [site Web Microsoft Store pour les entreprises](https://go.microsoft.com/fwlink/?linkid=2139192). 

Tous les autres clients accèdent à HUP à l’aide de cette étape :

- Dans le centre d’administration Microsoft, accédez à la page **facturation**  >  [**de vos produits** ](https://go.microsoft.com/fwlink/p/?linkid=842054), sélectionnez l’onglet **avantages** , puis sélectionnez l’onglet **HUP** .

## <a name="what-is-the-microsoft-home-use-program"></a>Qu’est-ce que le programme d’utilisation domestique de Microsoft ?

Le programme d’utilisation à la maison de Microsoft (HUP) permet aux employés éligibles d’acheter des produits grand public Microsoft à un prix réduit. Vous pouvez vérifier les remises actuelles pour chaque offre de produit en visitant la [page utiliser le produit programme use Home](https://www.microsoft.com/home-use-program).

## <a name="in-which-markets-is-the-home-use-program-offered"></a>Dans quels pays le programme d’utilisation à domicile est-il proposé ?

Les réductions sur les abonnements personnels et de la famille Microsoft 365 sont proposées aux entreprises éligibles dans le monde entier avec les exceptions suivantes :

- Macao, Russie et Venezuela : cette offre n’est pas disponible sur ces marchés.
- Chine et Hong Kong : un abonnement d’un an est disponible sur ces marchés. Pour la Chine, visitez le site [Microsoft 365 en Chine](https://www.microsoftstore.com.cn/home-use-program/invite). Pour Hong Kong, rendez-vous [sur l’achat de Microsoft 365 à un prix réduit](https://www.microsoftestore.com.hk/partner/hup?locale=en_HK).
- Japon : la famille Microsoft 365 n’est pas disponible.

## <a name="how-does-an-organization-qualify-for-hup-discounts"></a>Comment une organisation bénéficie-t-elle des remises HUP ?

Pour que les employés puissent tirer parti de la remise de programme Microsoft Home Use, ils ont besoin d’une adresse de messagerie d’entreprise valide et d’un compte Microsoft actif, et votre organisation doit répondre à l’un des critères suivants :

- Clients disposant d’une assurance logicielle active pour les applications Office éligibles 

- Clients ayant Microsoft 365/Microsoft 365 E3/E5 de Software Assurance

- Clients commerciaux et gouvernementaux ayant acheté au moins une licence de ≥ 2 000 ou une combinaison de Microsoft 365 E3/E5

- Clients éducation et non lucratif

## <a name="why-do-i-have-to-create-an-azure-active-directory-aad-account-to-use-the-home-use-program-microsoft-store-for-business"></a>Pourquoi ai-je besoin de créer un compte Azure Active Directory (AAD) pour utiliser le programme d’utilisation domestique/Microsoft Store pour les entreprises ?

Pour mettre à jour et moderniser le programme d’utilisation à domicile avec un large éventail d’avantages en termes de licence et de produit, nous devions modifier la façon dont nous effectuons HUP. Microsoft Store pour les entreprises est utilisé pour la validation du droit et pour attribuer des avantages HUP, et pour cela, il s’appuie sur AAD pour connecter des informations à partir de VSLC et pour valider les domaines de messagerie sur lesquels se trouvent les employés de votre organisation. Pour plus d’informations, consultez la rubrique [pour s’inscrire aux comptes Azure ad via Office 365 pour les entreprises](https://docs.microsoft.com/microsoft-store/sign-up-microsoft-store-for-business#o365-welcome).

## <a name="how-do-i-become-the-assigned-it-admin-of-a-managed-tenant"></a>Comment devenir l’administrateur informatique affecté à un client géré ?

Vous devez créer un enregistrement spécial dans le système DNS pour prouver que vous êtes propriétaire du domaine de votre entreprise, ex :contoso. com.

Pour obtenir une procédure pas à pas pour devenir un locataire géré, consultez la rubrique [prendre le relais d’un annuaire non géré en tant qu’administrateur dans Azure active](https://docs.microsoft.com/azure/active-directory/users-groups-roles/domains-admin-takeover).

## <a name="what-resources-are-available-to-help-promote-the-home-use-program-benefit-to-employees"></a>Quelles sont les ressources disponibles pour aider à promouvoir le programme d’utilisation domestique pour les employés ?

Utilisez les ressources de la page d’accueil du [programme use Home](https://www.microsoft.com/home-use-program/resources) pour vous aider à promouvoir le programme pour vos employés. Vous trouverez des prospectus numériques et imprimables, une formulation à utiliser dans les e-mails et les bulletins d’informations, ainsi que des conseils sur la façon de promouvoir au mieux le programme d’utilisation domestique.

Vous pouvez accéder à des documents d’intégration en plusieurs langues sur [HUP pour les clients](https://microsofteur.sharepoint.com/teams/HUPMaterial). Sélectionnez une langue pour commencer à télécharger les fichiers. Le fichier contient les ressources que vous pouvez utiliser pour sensibiliser vos employés au programme et les promouvoir en interne dans votre entreprise.

## <a name="what-are-the-steps-for-an-eligible-employee-to-make-a-microsoft-home-use-program-purchase"></a>Quelles sont les étapes à suivre pour qu’un employé éligible puisse effectuer un achat de programme à la maison Microsoft ?

1. **Inscription** Accédez à la [page utilisation de Microsoft Home Program](https://www.microsoft.com/home-use-program)et sélectionnez le bouton prise en main. Les employés entrent leur adresse de messagerie d’entreprise.

2. **Confirmer**  Un message électronique est envoyé à sa boîte de réception d’adresse de messagerie d’entreprise avec un lien qui authentifie son éligibilité pour le programme.

3. **Connexion** Une fois qu’ils ont été authentifiés, ils commencent le processus d’achat en fournissant le compte Microsoft associé à leur adresse de messagerie personnelle, ce qui leur permet d’afficher les remises sur les produits disponibles en fonction de l’avantage du programme d’utilisation domestique de l’organisation. Pour plus d’informations, consultez la rubrique relative à [la connexion à votre compte Microsoft](https://support.microsoft.com/help/4028195/microsoft-account-sign-in).

4. **Acheter** L’achat est traité et terminé dans le Microsoft Store.

## <a name="if-an-employee-subscribes-to-microsoft-365-through-the-microsoft-home-use-program-how-does-this-impact-our-organizations-microsoft-365-business-subscription-and-device-limit"></a>Si un employé s’abonne à Microsoft 365 par le biais du programme d’utilisation à la maison de Microsoft, quel impact cela a-t-il sur le nombre d’abonnements et de périphériques Microsoft 365 Business de votre organisation ?

Les installations via le programme d’utilisation domestique n’ont pas d’impact sur l’abonnement Microsoft 365 Business de votre organisation. Lorsque les employés activent Microsoft 365 sur leurs appareils personnels, ils utilisent leur compte Microsoft personnel, et non leur compte professionnel. Les administrateurs informatiques de votre organisation ne gèrent pas ou ne prennent pas en charge les achats effectués via le programme d’utilisation domestique.

## <a name="im-having-trouble-managing-my-organizations-microsoft-home-use-program-benefit-where-can-i-get-support"></a>Je ne parviens pas à gérer l’avantage du programme d’utilisation domestique de mon organisation. Où puis-je obtenir de l’aide ?

Pour prendre en charge les problèmes de migration à partir du centre de gestion des licences en volume ou pour accéder à Microsoft Store pour les entreprises, consultez la rubrique [prise en charge VLSC](https://www.microsoft.com/Licensing/servicecenter/default.aspx?wa=wsignin1.0).

Pour les problèmes d’affichage de l’avantage une fois que vous avez déjà accédé à Microsoft Store pour les entreprises, consultez l’aide de l' [administrateur Store pour les entreprises](https://docs.microsoft.com/microsoft-store/) .

## <a name="why-am-i-not-seeing-an-option-to-purchase-visio-professional-2019-andor-project-professional-2019"></a>Pourquoi ne puis-je pas voir une option permettant d’acheter Visio professionnel 2019 et/ou Project Professional 2019 ?

Si l’un de vos employés tire parti du programme précédent et recherche des réductions sur Visio Professional 2019 et/ou Project Professional 2019, suivez le processus de validation de droit et, une fois connecté avec votre compte Microsoft personnel, accédez à d' **autres produits Office** pour acheter ces produits. Si vous ne voyez pas les remises prévues, contactez l’administrateur de programme de votre employeur pour en savoir plus sur ces avantages supplémentaires.

- [Forum aux questions sur les employés du programme d’utilisation](https://www.microsoft.com/home-use-program/frequently-asked-questions)

 Pour obtenir de l’aide sur les problèmes techniques, consultez le [support Microsoft](https://support.microsoft.com/). Vous trouverez de l’aide pour :

- Statut de la commande

- Facturation

- Installation d’Office

- Connexion à Office
