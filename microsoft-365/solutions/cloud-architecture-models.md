# <a name="microsoft-cloud-for-enterprise-architects-illustrations"></a>Illustrations de documents sur le cloud Microsoft pour les architectes d’entreprise

Ces affiches sur l’architecture Cloud vous fournissent des informations sur les services de Cloud Computing Microsoft, notamment Microsoft 365, Azure Active Directory (Azure AD), Microsoft Intune, Microsoft Dynamics 365 et les solutions hybrides locales et de Cloud. 

Les décideurs et architectes informatiques peuvent utiliser ces ressources pour déterminer les solutions idéales pour leurs charges de travail et prendre des décisions concernant les composants d’infrastructure principaux, tels que la mise en réseau, l’identité et la sécurité.
