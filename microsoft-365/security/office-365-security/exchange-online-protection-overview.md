2. <span data-ttu-id="5a84b-121">Le message est ensuite inspecté à la recherche de programmes malveillants.</span><span class="sxs-lookup"><span data-stu-id="5a84b-121">Then the message is inspected for malware.</span></span> <span data-ttu-id="5a84b-122">Si un programme malveillant est détecté dans le message ou la ou les pièces jointes, le message est acheminé vers une mise en quarantaine de l’administrateur uniquement.</span><span class="sxs-lookup"><span data-stu-id="5a84b-122">If malware is found in the message or the attachment(s) the message is routed to an admin only quarantine store.</span></span> <span data-ttu-id="5a84b-123">Pour en savoir plus sur la protection contre les programmes malveillants, consultez la [protection contre les programmes malveillants dans EOP.](anti-malware-protection.md)</span><span class="sxs-lookup"><span data-stu-id="5a84b-123">To learn more about malware protection, see [Anti-malware protection in EOP](anti-malware-protection.md).</span></span>
