# <a name="manage-recipients-in-standalone-eop"></a>Gérer les destinataires dans EOP autonome

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**S’applique à**
-  [Exchange Online Protection autonome](exchange-online-protection-overview.md)

Les organisations Exchange Online Protection autonomes (EOP) sans boîtes aux lettres Exchange Online peuvent prendre en charge les types de destinataires suivants :

- **Utilisateurs de messagerie**: les utilisateurs de messagerie sont le type fondamental de comptes d’utilisateur dans votre organisation EOP autonome. Les utilisateurs de messagerie ont des informations d’identification de connexion dans votre organisation EOP, mais ils ont des adresses de messagerie externes (leurs boîtes aux lettres sont situées en dehors de votre organisation EOP).

  Pour plus d'informations sur la gestion des utilisateurs de messagerie dans EOP, consultez la rubrique [Gestion des utilisateurs de messagerie dans EOP](manage-mail-users-in-eop.md).

- **Groupes**: vous pouvez créer les types de groupes suivants :

  - Groupes de distribution
  - Groupes de sécurité à extension messagerie

  Pour plus d'informations sur la gestion des groupes dans EOP, consultez la rubrique [Gestion des groupes dans Exchange Online Protection (EOP)](manage-groups-in-eop.md).
