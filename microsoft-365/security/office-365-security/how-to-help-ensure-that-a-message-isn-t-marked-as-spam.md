# <a name="how-to-help-ensure-that-a-message-isnt-marked-as-spam"></a>Comment s’assurer qu’un message n’est pas marqué comme courrier indésirable

En tant qu'administrateur Exchange Online ou Exchange Online Protection (EOP), vous pouvez faire en sorte qu'un message électronique circulant via le service ne soit pas marqué comme courrier indésirable. Consultez la version mise à jour des conseils et procédures pour effectuer cette procédure dans [créer des listes d’expéditeurs approuvés dans Office 365](create-safe-sender-lists-in-office-365.md).
