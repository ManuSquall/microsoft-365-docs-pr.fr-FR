# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>Générer une alerte de test dans votre laboratoire d Microsoft 365 Defender d’évaluation  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**S’applique à :**
- Microsoft 365 Defender

Maintenant que vous avez terminé la configuration et la configuration de votre laboratoire d’évaluation Microsoft 365 Defender, vous pouvez exécuter une simulation d’attaque contre les menaces pour découvrir les fonctionnalités de détection, de corrélation d’alertes et de correction de réparation automatique de la solution de sécurité intégrée.  

1. Connectez-vous à https://security.microsoft.com
2. Dans la page d’accueil, recherchez une vignette intitulée **Simulation**.  Examinez le guide de procédure pas à pas pour savoir si vous pouvez créer un incident de test Microsoft 365 Defender.
