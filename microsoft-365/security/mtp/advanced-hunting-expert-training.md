<span data-ttu-id="b77dc-150">L’extrait suivant d’un fichier CSL présente un ensemble complet de conseils marqués comme commentaires avec `//` .</span><span class="sxs-lookup"><span data-stu-id="b77dc-150">The following excerpt from a CSL file shows a comprehensive set of guidance marked as comments with `//`.</span></span>

```kusto

// - Account information associated with the logon
// - The device which the account logged onto
// - The process which performed the logon
// - Network information (for network logons)
// - Timestamp
```

DeviceLogonEvents
| count

// DeviceLogonEvents
