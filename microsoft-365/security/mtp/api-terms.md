## <a name="apis"></a>API

 Les API Microsoft 365 Defender sont régies par la [licence et les conditions d’utilisation de l’API Microsoft](https://docs.microsoft.com/legal/microsoft-apis/terms-of-use).

### <a name="throttling-limits"></a>Limitations

Chaque API a ses propres limites de limitation.<br>
Il y a des limites de limitation par heure et par minute.<br>
Par exemple, les limites de limitation de l’API des incidents sont les suivantes : 50 appels par minute et 1500 par heure.<br>
Le code d’état de réponse HTTP en cas de limitation sera 429.
