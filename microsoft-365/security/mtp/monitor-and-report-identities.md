# <a name="identity-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Surveillance des identités et rapports dans le Centre de sécurité Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


Surveillez les identités de votre organisation et suivez les comportements suspects ou risqués. Dans la **catégorie Identités** des **rapports,** vous pouvez suivre :

* Comptes d’utilisateur présentant le plus d’anomalies
* Nombre d’utilisateurs signalés à risque par les stratégies d’accès conditionnel
* Nombre d’administrateurs généraux dans votre organisation

![Catégorie d’identités de la page de rapports](../../media/identities.png)

Pour les utilisateurs avec des détections spécifiques, vous pouvez explorer l’alerte spécifique et examiner dans le Centre de sécurité Microsoft Defender. Les détections incluent des anomalies telles que les utilisateurs qui se connectent à partir d’emplacements inconnus.

Pour obtenir un ensemble complet d’événements de risque, consultez Les événements de [risque Azure Active Directory.](https://docs.microsoft.com/azure/active-directory/reports-monitoring/concept-risk-events)
