# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>Générer une alerte de test dans votre atelier d’évaluation Microsoft 365 Defender  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**S’applique à :**
- Microsoft 365 Defender

À présent que vous avez terminé votre installation et la configuration du laboratoire d’évaluation de Microsoft 365 Defender, vous pouvez exécuter une simulation d’attaque de menace pour découvrir les fonctionnalités de détection, de corrélation des alertes et de correction spontanée de la solution de sécurité intégrée.  

1. Connectez-vous à https://security.microsoft.com
2. Sur la page d’accueil, recherchez une vignette intitulée **simulation**.  Consultez la procédure pas à pas pour créer un incident de test dans Microsoft 365 Defender.
<br>![Image de la carte de simulation dans le tableau de bord de sécurité Microsoft 365](../../media/mtp-eval-73.png) <br>
