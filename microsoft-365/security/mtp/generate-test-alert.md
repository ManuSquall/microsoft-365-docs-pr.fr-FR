# <a name="generate-a-test-alert-in-your-microsoft-threat-protection-evaluation-lab"></a>Générer une alerte de test dans votre atelier d’évaluation de la protection contre les menaces Microsoft  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**S’applique à :**
- Protection Microsoft contre les menaces

À présent que vous avez terminé l’installation et la configuration de votre laboratoire d’évaluation de la protection contre les menaces Microsoft, vous pouvez exécuter une simulation d’attaque de menace pour découvrir les fonctionnalités de détection, de corrélation des alertes et de correction spontanée de la solution de sécurité intégrée.  

1. Connectez-vous à https://security.microsoft.com
2. Sur la page d’accueil, recherchez une vignette intitulée **simulation**.  Consultez le Guide de démonstration pour obtenir la procédure à suivre pour créer un incident de test dans Microsoft Threat Protection.
<br>![Image de la carte de simulation dans le tableau de bord de sécurité Microsoft 365](../../media/mtp-eval-73.png) <br>
