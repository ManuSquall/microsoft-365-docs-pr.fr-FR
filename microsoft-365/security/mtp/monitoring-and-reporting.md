# <a name="monitor-and-view-reports-in-the-microsoft-365-security-center"></a>Surveiller et afficher des rapports dans le Centre de sécurité Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

> Vous souhaitez découvrir Microsoft 365 Defender ? Vous pouvez [l’évaluer dans un environnement de laboratoire](https://aka.ms/mtp-trial-lab) ou exécuter votre projet pilote en [production.](https://aka.ms/m365d-pilotplaybook)
>

Le Centre de sécurité Microsoft 365 fournit un résumé des statuts de protection et de sécurité au sein de votre environnement Microsoft 365.

Le centre de sécurité inclut une section **Rapports** qui propose une multitude de cartes couvrant différents domaines. Les analystes et les administrateurs de sécurité peuvent suivre les cartes dans le cadre de leurs opérations quotidiennes. Lors de l’analyse, les cartes fournissent des rapports détaillés et, dans certains cas, des options de gestion.

## <a name="customize-views"></a>Personnaliser les affichages

Par défaut, les cartes sont regroupées dans les catégories suivantes :
  
* [Identités :](monitor-and-report-identities.md) comptes d’utilisateur et informations d’identification
* [Données :](monitor-data.md) contenu du courrier électronique et du document
* [Appareils](monitor-devices.md) : ordinateurs, téléphones mobiles et autres appareils
* [Applications :](monitor-apps.md) programmes et services en ligne connectés

Basculez vers **Grouper par rubrique,** pour réorganiser les cartes et les grouper dans les rubriques suivantes :

* **Risque** : cartes qui mettent en évidence les entités, telles que les comptes et les appareils, qui peuvent être à risque. Ces cartes mettent également en évidence les sources de risque possibles, telles que les nouvelles campagnes contre les menaces et les applications cloud privilégiées  
* **Tendances de détection** : cartes qui mettent en évidence les nouvelles détections de menaces, les anomalies et les violations de stratégie
* **Configuration et santé :** cartes qui couvrent la configuration et le déploiement des contrôles de sécurité, y compris les états d’intégration d’appareil aux services de gestion
* **Autre** - toutes les autres cartes non classées dans d’autres rubriques
