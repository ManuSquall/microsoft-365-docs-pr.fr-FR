# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Surveillance des données et rapports dans le Centre de sécurité Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


La catégorie **Données** permet de suivre l’activité des utilisateurs susceptible d’entraîner la divulgation non autorisée de données. Il s’agit de la retravail des rapports de stratégie DLP existants et d’un rapport de correspondance de stratégie DLP tiers.

Vous pouvez voir les informations suivantes :

- Utilisateurs partageant le plus grand nombre de fichiers à partir d’applications cloud
- Nombre de correspondances de stratégie DLP
- Nombre de remplacements de stratégies DLP ou de faux positifs signalés
- Nombre de correspondances de stratégie DLP qui se sont produites dans des services cloud tiers via Microsoft Cloud App Security
