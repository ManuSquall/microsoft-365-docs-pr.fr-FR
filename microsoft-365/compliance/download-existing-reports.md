# <a name="download-existing-reports-in-the-security-amp-compliance-center"></a>Télécharger des rapports existants dans le &amp; Centre de sécurité conformité

Dans le [Centre de sécurité &amp; conformité](https://protection.office.com), plusieurs [rapports et](../security/office-365-security/reports-and-insights-in-security-and-compliance.md) informations sont disponibles pour aider l’équipe de sécurité de votre organisation à réduire et à résoudre les menaces pesant sur votre organisation. Si vous êtes membre de l’équipe de sécurité de votre organisation, vous pouvez télécharger un ou plusieurs rapports existants. 
  
## <a name="download-existing-reports"></a>Télécharger des rapports existants

> [!IMPORTANT]
> Assurez-vous que vous disposez des [autorisations nécessaires dans le &amp; Centre de sécurité conformité](../security/office-365-security/protect-against-threats.md). En règle générale, les administrateurs généraux, les administrateurs de la sécurité et les lecteurs de sécurité peuvent accéder aux rapports dans le centre de sécurité &amp; conformité. 
  
1. Dans le [ &amp; Centre de sécurité conformité](https://protection.office.com), accédez **à rapports sur les rapports** \> **à télécharger**.

2. Sélectionnez un ou plusieurs éléments dans la liste.

3. Cliquez sur **Télécharger le rapport**, puis cliquez sur **Fermer**.
