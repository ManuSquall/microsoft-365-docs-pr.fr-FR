5. <span data-ttu-id="7cd02-173">Sur la page **Ajouter des informations d’identification pour le site SFTP de conversation Ice** , sous étape 3, entrez les informations requises dans les zones suivantes, puis cliquez sur **valider la connexion**.</span><span class="sxs-lookup"><span data-stu-id="7cd02-173">On the **Add credentials for ICE Chat SFTP site** page, under Step 3, enter the required information in the following boxes and then click **Validate connection**.</span></span>

   - <span data-ttu-id="7cd02-175">**Mot de passe :** Le mot de passe de votre site SFTP de conversation ICE.</span><span class="sxs-lookup"><span data-stu-id="7cd02-175">**Password:** The password for your ICE Chat SFTP site.</span></span>

   - <span data-ttu-id="7cd02-176">**URL sftp :** L’URL du site SFTP de conversation ICE (par exemple, sftp.theice.com).</span><span class="sxs-lookup"><span data-stu-id="7cd02-176">**SFTP URL:** The URL for the ICE Chat SFTP site (for example, sftp.theice.com).</span></span>

   - <span data-ttu-id="7cd02-177">**Port sftp :** Numéro de port du site SFTP de conversation ICE.</span><span class="sxs-lookup"><span data-stu-id="7cd02-177">**SFTP port:** The port number for the ICE Chat SFTP site.</span></span> <span data-ttu-id="7cd02-178">Le connecteur utilise ce port pour se connecter au site SFTP.</span><span class="sxs-lookup"><span data-stu-id="7cd02-178">The connector uses this port to connect to the SFTP site.</span></span>

7. <span data-ttu-id="7cd02-180">Sur la page **connecter des utilisateurs externes à des utilisateurs de Microsoft 365** , activez le mappage utilisateur automatique et fournissez les mappages utilisateur personnalisés requis.</span><span class="sxs-lookup"><span data-stu-id="7cd02-180">On the **Map external users to Microsoft 365 users** page, enable automatic user mapping and provide custom user mapping as required.</span></span> <span data-ttu-id="7cd02-181">Vous pouvez télécharger une copie du fichier CSV de mappage d’utilisateurs sur cette page.</span><span class="sxs-lookup"><span data-stu-id="7cd02-181">You can download a copy of the user-mapping CSV file on this page.</span></span> <span data-ttu-id="7cd02-182">Vous pouvez ajouter les mappages utilisateur au fichier, puis le télécharger.</span><span class="sxs-lookup"><span data-stu-id="7cd02-182">You can add the user mappings to the file and then upload it.</span></span>

   > [!NOTE]
   > <span data-ttu-id="7cd02-183">Comme expliqué précédemment, le fichier CSV de mappage de fichiers de correspondance contient le Imid de conversation ICE et l’adresse de boîte aux lettres Microsoft 365 correspondante pour chaque utilisateur.</span><span class="sxs-lookup"><span data-stu-id="7cd02-183">As previously explained, custom mapping file CSV file contains the ICE Chat imid and corresponding Microsoft 365 mailbox address for each user.</span></span> <span data-ttu-id="7cd02-184">Si vous activez le mappage utilisateur automatique et que vous fournissez un mappage personnalisé, pour chaque élément de conversation, le connecteur regarde d’abord le fichier de mappage personnalisé.</span><span class="sxs-lookup"><span data-stu-id="7cd02-184">If you enable automatic user mapping and provide a custom mapping, for every chat item, the connector will first look at custom mapping file.</span></span> <span data-ttu-id="7cd02-185">S’il ne trouve pas d’utilisateur Microsoft 365 valide correspondant au Imid de conversation ICE d’un utilisateur, le connecteur importe l’élément dans les boîtes aux lettres pour les utilisateurs spécifiés dans les propriétés *SenderEmail* et *RecipientEmail* de l’élément conversation.</span><span class="sxs-lookup"><span data-stu-id="7cd02-185">If it doesn't find a valid Microsoft 365 user that corresponds to a user's ICE Chat imid, the connector will import the item to the mailboxes for the users specified in the *SenderEmail* and *RecipientEmail* properties of the chat item.</span></span> <span data-ttu-id="7cd02-186">Si le connecteur ne trouve pas d’utilisateur valide de Microsoft 365 par le biais d’un mappage utilisateur automatique ou personnalisé, l’élément n’est pas importé.</span><span class="sxs-lookup"><span data-stu-id="7cd02-186">If the connector doesn't find a valid Microsoft 365 user by either automatic or custom user mapping, the item won't be imported.</span></span>
