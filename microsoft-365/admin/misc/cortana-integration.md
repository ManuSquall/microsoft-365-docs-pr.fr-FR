# <a name="cortana-in-microsoft-365"></a>Cortana dans Microsoft 365

Cortana, votre Assistant de productivité personnelle, propose des expériences dotées de fonctionnalités autonomes pour gagner du temps et attirer votre attention sur ce qui se présente le plus. Cortana aide vos utilisateurs à augmenter leur productivité personnelle pendant toute la durée de leur travail et de leur vie. Lorsqu’ils sont connectés avec des comptes professionnels ou scolaires valides, les utilisateurs peuvent obtenir des services d’assistance de Cortana en nuage dans les outils de productivité Microsoft 365. Cortana dans Microsoft 365 répond aux engagements de confidentialité, de sécurité et de conformité de Microsoft 365 à l’échelle de l’entreprise.

En tant qu’assistant de productivité personnelle, Cortana est conçue pour fournir des fonctionnalités qui traitent et justifient en toute sécurité et en toute sécurité les données Microsoft 365, par exemple les e-mails, les fichiers, les conversations, etc., pour gagner du temps, augmenter l’efficacité et améliorer la productivité de vos utilisateurs.

À l’avenir, nous concentrons Cortana sur la productivité de l’entreprise.

- Conformément aux autres services Microsoft 365, Cortana dans Microsoft 365 répond aux mêmes engagements de confidentialité, de sécurité et de conformité au niveau de l’entreprise, comme indiqué dans les [services en ligne (OST)](https://www.microsoft.com/en-us/licensing/product-licensing/products).

- Les nouveaux services d’assistance de Cortana basée sur le Cloud, tels que le courrier électronique de briefing et lire mes courriers électroniques, seront activés à l’aide de Cortana dans Microsoft 365 et sont entièrement conformes à ces promesses. Ces fonctionnalités sont actuellement disponibles sur le marché anglais (États-Unis) pour les utilisateurs disposant de boîtes aux lettres aux États-Unis et pour les employés de sociétés multinationales avec des boîtes aux lettres américaines.

- Les utilisateurs peuvent se connecter à Cortana dans les services Microsoft 365 décrits ici via des applications clientes, telles qu’Outlook Mobile, sous réserve de conditions de licence distinctes.

- Les expériences de consommation existantes, notamment Cortana dans Windows 10 (version 1909 et antérieures) et l’application Cortana sur iOS et Android, sont régies par le [contrat de services Microsoft](https://www.microsoft.com/en-us/licensing/product-licensing/products) et la [déclaration de confidentialité de Microsoft](https://privacy.microsoft.com/en-us/privacystatement). Ces termes régiront également Cortana dans les services Microsoft 365 fournis à l’utilisateur lorsqu’ils se connectent avec leurs informations d’identification consommateur.

## <a name="what-data-is-processed-by-cortana-in-microsoft-365"></a>Quelles sont les données traitées par Cortana dans Microsoft 365 ?

Dans Microsoft 365, Cortana traite les requêtes de l’utilisateur, les données Office nécessaires pour répondre à la demande de l’utilisateur, ainsi que d’autres télémétries générées par les systèmes Microsoft pour exécuter le service. Les données collectées par Cortana dans Microsoft 365 incluent la représentation textuelle des requêtes orales de l’utilisateur (par exemple, les transcriptions de la reconnaissance vocale). Ces données de texte sont des données client et sont gérées conformément aux [conditions des services en ligne](https://www.microsoft.com/en-us/licensing/product-licensing/products). Il est utilisé uniquement pour développer et améliorer les modèles d’apprentissage automatique conformes aux [conditions des services en ligne](https://www.microsoft.com/en-us/licensing/product-licensing/products).

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-in-microsoft-365"></a>Quel est le modèle de gouvernance pour les données client dans Cortana dans Microsoft 365 ?

Conformément aux autres services Microsoft 365 et Microsoft 365, Cortana dans Microsoft 365 est sécurisée et soumise aux [conditions des services en ligne](https://www.microsoft.com/en-us/licensing/product-licensing/products). Cela inclut un ensemble de promesses pour la protection des données du client contre la perte accidentelle, la modification, la divulgation ou l’accès non autorisé ou une destruction illicite. Les données client sont également sujettes à des limitations d’accès strictes. Microsoft utilise les données client uniquement pour fournir les services approuvés et à des fins compatibles avec ces services. Pour plus d’informations, consultez le tableau ci-dessous.

## <a name="cortana-in-microsoft-365-features"></a>Fonctionnalités de Cortana dans Microsoft 365

### <a name="play-my-emails"></a>Lire mes courriers électroniques

Lire mes courriers électroniques (en tant que connexion via Outlook Mobile) est une expérience pratique basée sur les communications vocales permettant aux utilisateurs d’écouter leur boîte de réception via les haut-parleurs sur leur téléphone, casque ou périphérique audio connecté. Les utilisateurs peuvent demander à Cortana de lire leurs messages récents à voix haute et demander à Cortana de prendre des mesures telles que l’indicateur, l’archivage, la suppression et l’omission de messages. Cette fonctionnalité est particulièrement utile pour rattraper votre courrier électronique lors de vos déplacements ou en déplacement. Lorsque l’utilisateur parle à Cortana dans lire mes courriers électroniques, la demande audio accède directement à Cortana dans Microsoft 365. Une lecture de texte sur la parole du courrier électronique de l’utilisateur est effectuée dans le Cloud Microsoft 365. Pendant ce processus, aucune donnée Microsoft 365 n’est traitée sur l’appareil mobile de l’utilisateur et aucune donnée de courrier électronique n’est enregistrée. Une transcription des commandes vocales (c’est-à-dire « marquer comme lu », « suivant », « indicateur », etc.) peut être conservée conformément aux conditions de protection des données dans les [conditions des services en ligne](https://www.microsoft.com/en-us/licensing/product-licensing/products).

Cortana appellera lorsqu’un courrier électronique est protégé et suspendra brièvement avant de lire le message afin de laisser aux utilisateurs suffisamment de temps pour suspendre la lecture ou passer au message suivant. À l’instar d’un appel téléphonique privé, les utilisateurs doivent faire preuve de prudence lors de l’initiation de la lecture dans les emplacements où des informations confidentielles peuvent potentiellement être survenues. Dans ce cas, il est recommandé aux employés de votre organisation d’utiliser un casque dans les environnements appropriés lors de l’utilisation de l’option lire mes courriers électroniques dans Outlook Mobile.

### <a name="opt-out-of-play-my-emails"></a>Désactivation de la lecture de mes courriers électroniques

Les utilisateurs peuvent choisir de lire mes courriers électroniques en suivant les étapes ci-dessous.

1. Ouvrez Outlook Mobile.

2. Accédez à **paramètres**.
  
3. Sélectionnez **lire mes courriers électroniques**.

4. Déplacez le bouton bascule sur désactivé sur les comptes que vous souhaitez désactiver.

### <a name="briefing-email"></a>Courrier électronique de briefing

Cortana envoie un message électronique de briefing personnalisé avec des tâches et des engagements que vous avez effectués avec un moyen pratique de les marquer comme étant **achevées** ou de planifier les délais de mise au point pour les faire. Elle inclut également un résumé de vos réunions et des documents pertinents pour votre journée. Cortana extrait les informations des messages Microsoft 365 d’un utilisateur et les stocke dans leur boîte aux lettres Microsoft 365 jusqu’à ce qu’elle soit consolidée en bref message électronique. À aucun moment, les données personnelles ne sont pas accessibles en dehors de votre boîte aux lettres Microsoft 365.

### <a name="how-to-opt-out-of-briefing-email"></a>Procédure de refus d’un message électronique de briefing

Les administrateurs peuvent configurer le briefing de leur organisation à l’aide d’un [bouton bascule](https://docs.microsoft.com/briefing/be-admin) dans le centre d’administration 365 de Microsoft. Les utilisateurs peuvent désactiver le courrier de briefing de Cortana en sélectionnant annuler l’abonnement dans le pied de page du message.

Nous continuerons à introduire de plus en plus de Cortana dans les fonctionnalités de Microsoft 365 pour améliorer la productivité de votre organisation.

En savoir plus sur les [offres de conformité Microsoft](https://docs.microsoft.com/microsoft-365/compliance/offering-home).

## <a name="how-is-cortana-in-microsoft-365-different-from-other-cortana-features-i-may-have-previously-experienced"></a>En quoi la fonctionnalité Cortana dans Microsoft 365 est-elle différente des autres fonctionnalités de Cortana que j’ai peut-être déjà expérimentées ?

Voici les deux façons de penser au fonctionnement de Cortana dans votre entreprise :

**Nouvelles expériences pour les organisations avec Cortana dans microsoft 365**: les services Cortana dans Microsoft 365 sont conçus pour répondre aux besoins des organisations en matière de sécurité et de conformité.

1. Il s’agit d’un nouveau service qui est abordé ici dans ce document.

2. Pour les services soumis aux conditions des services en ligne, Microsoft est un processeur de données : Microsoft recueille et utilise les données client des clients uniquement pour fournir les services en ligne demandés par nos clients et pour les besoins de nos clients. Dans le règlement général sur la protection des données (RGPD) de l’UE, le client est le contrôleur de données de ses données. Consultez les [conditions des services en ligne](https://www.microsoft.com/en-us/licensing/product-licensing/products) et [Présentation de la transparence de la confidentialité pour nos clients du nuage commercial](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/).

3. Par exemple, lire mes courriers électroniques est un service Cortana auquel les utilisateurs peuvent se connecter via Outlook pour iOS et utilise Cortana dans Microsoft 365.

4. Les administrateurs disposent toujours de contrôles pour les expériences connectées facultatives pour Cortana, similaires aux expériences en connexion facultatives lors de l’utilisation d’applications Office ProPlus.

**Services existants pour les consommateurs**: les services connectés facultatifs de Cortana sont conçus principalement pour les expériences de consommation et sont actuellement fournis dans Windows 10 (version 1909 et antérieures) et l’application Cortana sur iOS et Android.

1. Ces expériences activent des fonctionnalités telles que la météo, les actualités et le trafic.

2. Les administrateurs clients peuvent contrôler si Cortana dans Windows 10 (version 1909 et antérieures) et l’application Cortana sur iOS et Android peuvent autoriser Cortana à se connecter aux données client Microsoft 365.

Désactiver l’accès Cortana aux données hébergées par Microsoft de votre organisation

1. Dans le centre d’administration 365 de Microsoft, sélectionnez **paramètres** > **services** et sélectionnez **Cortana**.

2. Désélectionnez la case à cocher **autoriser ces expériences de connexion facultatives Cortana à utiliser les données hébergées par Microsoft de votre organisation** pour désactiver les expériences connectées à Cortana.

3. Sélectionnez **Enregistrer les modifications**.

Pour les services régis par le [contrat de services Microsoft](https://go.microsoft.com/fwlink/p/?LinkId=2109174) et la [déclaration de confidentialité Microsoft](https://privacy.microsoft.com/en-us/privacystatement), Microsoft est le contrôleur de données. En tant que contrôleur de données, Microsoft utilise des données pour améliorer les produits et les services conformément à la [déclaration de confidentialité de Microsoft](https://privacy.microsoft.com/en-us/privacystatement).
