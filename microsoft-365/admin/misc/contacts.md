# <a name="quick-help-contacts"></a>Aide rapide : contacts

Si vous avez besoin de contacts partagés auxquels tous les membres de votre organisation peuvent accéder, créez-les dans le centre d’administration 365 de Microsoft. Les contacts que vous créez ici peuvent être affichés par vous et vos utilisateurs dans le cadre de la liste d'adresses globale.
  
## <a name="how-do-i-add-contacts-in-the-admin-center"></a>Comment ajouter des contacts dans le centre d'administration ?

1. Dans le centre d’administration, accédez à la page des <a href="https://go.microsoft.com/fwlink/p/?linkid=2053302" target="_blank">contacts</a> de l' **utilisateur** \> .

2. Sur la page **contacts** , sélectionnez **Ajouter un contact**.
  
3. Sur la page **nouveau contact** , renseignez les détails et sélectionnez **Ajouter** pour créer un contact.
  
## <a name="how-are-these-contacts-different-from-my-contacts"></a>En quoi ces contacts sont-ils différents de mes contacts ?

Mes contacts sont des contacts que vous créez pour vous-même et que vos utilisateurs créent pour eux-mêmes, mais d’autres non. Les contacts que vous créez dans le centre d’administration sont des contacts de l’organisation et tout le monde peut les afficher en plus de leur propre. Pour en savoir plus, consultez la rubrique [vue d’ensemble des personnes-Outlook Web App](https://support.microsoft.com/en-us/office/people-overview-outlook-web-app-5fe173cf-e620-4f62-9bf6-da5041f651bf).
  
## <a name="how-does-everyone-get-to-the-contacts-i-created-in-the-admin-center"></a>Comment les membres de l'organisation peuvent-ils accéder aux contacts que j'ai créés dans le centre d'administration ?

 Ils peuvent accéder à **personnes** dans Microsoft 365, développer **répertoire** et sélectionner **tous les contacts**. Ils peuvent afficher chaque contact et leurs informations à partir de cet emplacement.
  
## <a name="can-anyone-create-and-edit-these-organizational-contacts"></a>Est-il possible de créer et modifier ces contacts de l'organisation ?

Non. Seuls les administrateurs **généraux** et **Exchange** peuvent créer, modifier ou supprimer ces contacts. Toutes les autres personnes peuvent seulement les afficher.
  
## <a name="can-i-use-this-to-manage-my-business-clients"></a>Puis-je gérer mes clients professionnels à l'aide de cette fonctionnalité ?

Vous pouvez utiliser les contacts Microsoft 365 comme vous le souhaitez, mais il existe des limitations. En savoir plus sur [les autres méthodes de gestion des contacts](ways-to-manage-contacts.md)
  
## <a name="how-do-i-bulk-import-organizational-contacts"></a>Comment importer en bloc des contacts professionnels ?

Utilisez Windows PowerShell et un fichier CSV (valeurs séparées par des virgules) pour importer en bloc des contacts externes comme décrit dans [Bulk import external contacts to Exchange Online](../../compliance/bulk-import-external-contacts.md).
