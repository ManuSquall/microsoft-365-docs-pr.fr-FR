# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Mettre à jour vos enregistrements MX pour effectuer la transition vers le service Exchange Online global

1. Connectez-vous au [portail d’administration de Microsoft 365](https://admin.microsoft.com), puis accédez à **paramètres** > **Domains**

2. Status s’affiche sur le côté droit de chaque domaine. Si les domaines de votre organisation pointent vers Microsoft Cloud Germany Exchange Online, vous devrez mettre à jour votre enregistrement MX.

3. Cliquez sur le domaine, puis sur **Erreurs DNS détectées, cliquez ici pour afficher**.

4. Cette page contient des instructions pour vous montrer comment corriger l’enregistrement MX. Si le serveur d’inscriptions de votre domaine utilise la [connexion au domaine](../setup/add-domain.md#registrars-with-domain-connect), vous pouvez cliquer sur « corriger mes enregistrements » en haut. Sinon, vous pouvez suivre le lien de l’Assistant pour **obtenir des instructions détaillées** pour votre serveur d’inscriptions.
