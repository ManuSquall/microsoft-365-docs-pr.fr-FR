# <a name="icann-verification-of-contact-information"></a>Vérification de l’ICANN des coordonnées

Lorsque vous achetez un domaine ou modifiez les informations de contact d’un domaine, le serveur d’inscriptions est tenu de valider vos informations de contact pour se conformer aux réglementations pour l’ICANN.

Pour la validation, un message électronique est envoyé à l’adresse de l’abonné.

 **Pour vérifier vos informations de contact, sélectionnez le lien dans le message électronique de vérification.** Si les propriétaires de domaine ne vérifient pas l’adresse de messagerie, le domaine est suspendu. Ensuite, accédez à la [page domaines](https://admin.microsoft.com/adminportal/home?ref=Domains) dans le centre d’administration et vérifiez l’état de votre domaine.
