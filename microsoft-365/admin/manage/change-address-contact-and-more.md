# <a name="change-your-organizations-address-technical-contact-and-more"></a>Modifier l'adresse de votre organisation, le contact technique et d'autres informations de profil
  
Vous pouvez apporter des modifications au profil de votre organisation, comme le nom, l’adresse, le téléphone et le contact technique de votre organisation. **Pour mettre à jour ces informations, vous devez être un administrateur général.**
  
Pour modifier l’adresse associée à votre facture ou abonnement, consultez [Modifier vos adresses de facturation pour Microsoft 365 pour les entreprises.](../../commerce/billing-and-payments/change-your-billing-addresses.md)

## <a name="edit-organization-information"></a>Modifier les informations de l'organisation

> [!IMPORTANT]
> Vous ne pouvez pas changer le pays ou la région pour votre abonnement. C’est parce que le pays ou la région où votre organisation a son siège social détermine quels services sont à votre disposition, les taxes et la monnaie de facturation, et l’emplacement du centre de données. Pour changer le pays ou la région de votre organisation, inscrivez-vous à un nouveau compte, choisissez le pays ou la région désiré et achetez un nouvel abonnement.

Pour modifier d’autres informations sur la page de profil de votre entreprise :
  
1. Dans le centre d’administration, rendez-vous **sur la** page Paramètres paramètres \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2053743" target="_blank">**Org.**</a>

2. Sur **l’onglet Profil de l’Organisation,** sélectionnez informations **organisation .**

3. Mettez à jour les informations de votre organisation, puis sélectionnez **Enregistrer les modifications**. Assurez-vous de remplir tous les champs requis marqués d’un * pour permettre d’enregistrer vos modifications.

Chaque champ est décrit ci-dessous.

## <a name="change-your-alternate-email-address"></a>Modifiez votre autre adresse e-mail

En tant qu’administrateur, vous pouvez modifier votre adresse e-mail alternative pour réinitialiser les mots de passe.

1. Allez au centre <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Microsoft 365 admin .</a>

2. Dans l’en-tête, sélectionnez votre icône de profil, puis sélectionnez **Afficher le compte**.

3. Sur la gauche, sélectionnez **Informations de sécurité**.

4. Dans la section E-mail, **sélectionnez Modifier**

5. Modifiez l’adresse dans la **boîte e-mail** Alternative, puis sélectionnez **Suivant**.

6. Entrez le code envoyé à l’adresse e-mail alternative et **sélectionnez OK** pour vérifier la modification.
Pour en savoir plus sur la modification d’autres informations de profil, [consultez Modifier vos préférences de contact ou](change-contact-preferences.md) modifier votre langage [d’affichage](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b.aspx).
