# <a name="manage-messages-in-message-center"></a>Gérer les messages dans le centre de messages

Les publications du centre de messages sont des annonces importantes concernant les fonctionnalités nouvelles et modifiées. Cet article fournit plus d’informations sur les choses que vous pouvez faire avec les messages pour faciliter la gestion des changements dans votre organisation.
  
## <a name="read-a-message"></a>Lire un message

Pour aller au Centre de messages, allez dans le centre [d’administration,](https://go.microsoft.com/fwlink/p/?linkid=2024339)puis sélectionnez **Centre de**  >  **messages d’état d’état.** 

Sélectionnez un titre de message pour l’ouvrir dans le volet de lecture. Chaque message contient un résumé de la modification, une déclaration de la façon dont la modification peut vous affecter, des informations sur la façon dont vous pouvez préparer votre organisation pour la modification et (dans la plupart des cas) un lien d’informations supplémentaire dans lequel vous pouvez en savoir plus. Les informations supplémentaires restent disponibles, même après l’expiration d’un message.

## <a name="give-feedback-on-a-post"></a>Donner des commentaires sur un billet

Dans le centre de messages, vous pouvez sélectionner un message pour voir les détails.

Si vous souhaitez fournir des commentaires  sur  le message, sélectionnez l’icône J’aime ou N’aime pas et fournissez des commentaires dans la zone de texte qui s’affiche. Ne fournissez aucune information personnelle. Vous pouvez éventuellement sélectionner **C’est ok pour me contacter** à propos de ce commentaire, puis sélectionner **Envoyer.**

## <a name="share-a-message"></a>Partager un message

Vous voyez un message sur le fait que quelqu’un d’autre doit agir ? Vous pouvez partager le contenu du message avec n’importe quel utilisateur par courrier électronique :
  
1. Sélectionnez le message pour l’ouvrir, puis sélectionnez **Partager.**
  
2. Pour partager le message, entrez jusqu’à deux adresses de messagerie séparées par un deux-points. Vous pouvez envoyer des messages à des adresses de messagerie individuelles et à des groupes. Si vous le souhaitez, vous pouvez choisir de recevoir une copie du message par courrier électronique (le message sera envoyé à votre adresse de messagerie principale) ou d’ajouter un message personnel pour fournir aux destinataires davantage de contexte.
  
3. Sélectionnez **Partager** pour envoyer le courrier électronique.
    
## <a name="get-a-link"></a>Obtenir un lien

Vous devez suivre un autre administrateur pour vous assurer qu’il est informé d’une modification et prendre des mesures ? Vous pouvez générer un lien à partager dans le courrier électronique ou la messagerie instantanée, par exemple, qui connectera l’utilisateur directement à ce message. La personne avec qui vous partagez le lien doit être administrateur dans Microsoft 365 pour votre organisation, sinon elle n’aura pas accès au message lié.

::: moniker range="o365-worldwide"

1. Sélectionnez le message pour l’ouvrir.

2. Sélectionnez le **lien Copier dans le Presse-papiers** (lien).

3. Autorisez la page web à accéder à votre Presse-papiers. Un lien direct est copié dans le Presse-papiers.

4. Utilisez Ctrl+V ou cliquez avec le bouton droit et **sélectionnez Coller** pour insérer le lien.

::: moniker-end

::: moniker range="o365-germany"

1. Sélectionnez le message pour l’ouvrir.

2. Choose **More** \> **Get a link** from the action bar.

3. Autorisez la page web à accéder à votre Presse-papiers. Un lien direct est copié dans le Presse-papiers.

4. Utilisez Ctrl+V ou cliquez avec le bouton droit et **sélectionnez Coller** pour insérer le lien.

::: moniker-end

::: moniker range="o365-21vianet"

1. Sélectionnez le message pour l’ouvrir.

2. Choose **More** \> **Get a link** from the action bar.

3. Autorisez la page web à accéder à votre Presse-papiers. Un lien direct est copié dans le Presse-papiers.

4. Utilisez Ctrl+V ou cliquez avec le bouton droit et **sélectionnez Coller** pour insérer le lien.

::: moniker-end

## <a name="read-and-unread-states"></a>États de lecture et non lus

Tous les messages non lus dans le centre de messages s’affichent en gras. L’ouverture d’un message le marque comme lu. Vous pouvez marquer un message comme non lu.

::: moniker range="o365-worldwide"

1. Sur la page principale du centre de messages, sélectionnez un message.

2. Sélectionnez **l’icône Marquer** comme non lu (enveloppe) en haut du message.
  
::: moniker-end

::: moniker range="o365-germany"

1. Sélectionnez le message pour l’ouvrir.

2. Sélectionnez **Plus** \> **de marques comme non lues.**

3. Fermez le message.

::: moniker-end

::: moniker range="o365-21vianet"

1. Sélectionnez le message pour l’ouvrir.

2. Sélectionnez **Plus** \> **de marques comme non lues.**

3. Fermez le message.

::: moniker-end

## <a name="archive-and-restore"></a>Archiver et restaurer

Si vous voyez un message qui ne vous concerne pas, ou si vous avez peut-être déjà agir dessus, vous pouvez archiver le message pour le supprimer de l’affichage Centre de messages. L’affichage que vous voyez dans le centre de messages est spécifique à votre compte d’utilisateur, donc son archivage à partir de votre affichage n’affecte pas les autres administrateurs.

::: moniker range="o365-worldwide"

- Sur la page principale du centre de messages, sélectionnez un message, puis sélectionnez **Archiver** au-dessus de la liste des messages.

- Ouvrez le message, puis sélectionnez **Archive** en bas du message.

Vous avez besoin de récupérer un message archivé ? Pas de problème.
  
1. Sélectionnez **l’onglet Messages archivés** en haut du centre de messages. Une liste de messages archivés s’affiche. 

2. Sélectionnez le message, **sélectionnez Restaurer** et le message est restauré dans les messages actifs.

::: moniker-end

::: moniker range="o365-germany"

Il existe deux façons d’archiver un message.
  
- Sélectionnez le X qui apparaît à droite de l’ID de message. Le message est supprimé de votre liste. Or,

- Ouvrez le message, puis sélectionnez **Archive** en haut du message.

Vous avez besoin de récupérer un message rejeté ? Pas de problème.
  
1. Sélectionnez **le filtre Messages archivés** dans le menu déroulant **Affichages.** Une liste de messages archivés s’affiche.

2. **Sélectionnez** Restaurer à droite de l’ID du message et le message est restauré dans les messages actifs.

::: moniker-end

::: moniker range="o365-21vianet"

Il existe deux façons d’archiver un message.
  
- Sélectionnez le X qui apparaît à droite de l’ID de message. Le message est supprimé de votre liste. Or,

- Ouvrez le message, puis sélectionnez **Archive** en haut du message.

Vous avez besoin de récupérer un message archivé ? Pas de problème.
  
1. Sélectionnez **le filtre Messages archivés** dans le menu déroulant **Affichages.** Une liste de messages archivés s’affiche.

2. **Sélectionnez** Restaurer à droite de l’ID de message et le message est restauré dans les messages actifs.

::: moniker-end

Il peut être difficile de suivre les modifications qui nécessitent l’exécution des tâches, quand et par qui, et de suivre chaque tâche jusqu’à son achèvement. Vous pouvez également noter quelque chose et l'étiqueter pour le vérifier plus tard. Vous pouvez faire tout cela et bien plus encore lorsque vous synchronisez vos messages du Centre d’administration Microsoft 365 avec le Planificateur Microsoft. Pour plus d’informations, [voir Suivre les tâches de votre centre de messages dans le Planificateur.](https://docs.microsoft.com/office365/planner/track-message-center-tasks-planner)
