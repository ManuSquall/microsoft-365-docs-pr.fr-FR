# <a name="let-users-reset-their-own-passwords"></a>Autoriser les utilisateurs à réinitialiser leurs mots de passe

En tant qu Microsoft 365 administrateur, vous pouvez laisser les gens utiliser [l’outil de réinitialisation de mot de passe libre-service](https://go.microsoft.com/fwlink/p/?LinkId=522677) afin que vous n’avez pas à réinitialiser les mots de passe pour eux. Moins de travail pour vous !
  
## <a name="before-you-begin"></a>Avant de commencer
  
- Vous obtenez la réinitialisation gratuite du mot de passe libre-service pour les **utilisateurs** du cloud avec n’importe quel Microsoft 365, de l’éducation ou du plan payant à but non lucratif. Ça ne marche pas avec Microsoft 365 procès.

- Elle utilise Azure. Vous bénéficierez automatiquement et **gratuitement** de cette fonction dans Azure lorsque vous effectuerez ces étapes. L'activation de la réinitialisation du mot de passe libre-service ne vous coûtera rien si vous n'utilisez aucune autre fonctionnalité Azure.

- **Si vous utilisez un répertoire actif sur place, les deux** points ci-dessus ne s’appliquent pas. Au contraire, vous pouvez configurer **cela, mais il nécessite un abonnement payant à Azure AD Premium**.

Cet article s’adresse aux personnes responsables de la stratégie d’expiration des mots de passe au sein d’une entreprise, d’une école ou d’une association. Pour effectuer ces étapes, vous devez vous connecter avec votre compte d’administrateur Microsoft 365. [Qu’est-ce qu’un compte admin ?](../../business-video/admin-center-overview.md)

Vous devez être un administrateur [global ou un administrateur de mot de](about-admin-roles.md) passe pour effectuer ces étapes.

## <a name="steps-let-people-reset-their-own-passwords"></a>Étapes : Laissez les gens réinitialiser leurs propres mots de passe

Ces étapes activent la réinitialisation du mot de passe libre-service pour tout le monde dans votre entreprise.

1. Dans le <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">centre admin</a>, allez à la page **Paramètres paramètres**  >  **Org.**

2. En haut de la page **paramètres Org,** sélectionnez l’onglet **Sécurité & confidentialité.**
  
3. Sélectionnez **Réinitialisation de mot de passe libre-service**.

4. Dans le **cadre de la réinitialisation du mot de passe libre-service,** **sélectionnez Rendez-vous sur le portail Azure pour activer la réinitialisation de mot de passe libre-service.**

5. Dans le volet de navigation gauche, **sélectionnez Utilisateurs,** puis, sur les **utilisateurs | Toutes les pages utilisateurs,** sélectionnez **Mot de passe réinitialiser**.
  
6. Sur la page **Propriétés,** sélectionnez **Tous pour** l’activer pour tous les membres de votre entreprise, puis sélectionnez **Enregistrer**.
  
7. Lorsque vos utilisateurs se connectent, ils seront invités à saisir des coordonnées supplémentaires qui les aideront à réinitialiser leur mot de passe à l’avenir.
