# <a name="pre-requisites-for-protecting-data-on-devices-with-microsoft-365-business"></a>Conditions préalables à la protection des données sur les appareils avec Microsoft 365 Business

[] La première étape de la configuration de Microsoft 365 Entreprise pour votre organisation consiste à vérifier que vous êtes en mesure de remplir les conditions préalables.
  
## <a name="pre-requisites"></a>Conditions préalables

- Les appareils Windows doivent exécuter Windows 7 professionnel, Windows 8 professionnel ou Windows 8,1 Pro.
    
    [Mettre à niveau les appareils Windows vers Windows Professionnel Creators Update](upgrade-to-windows-pro-creators-update.md)
    
- Supprimez les appareils des solutions de gestion mobile (MobileIron, AirWatch, etc.). Vous allez inscrire toutes les personnes de votre organisation à la gestion mobile Microsoft 365 Entreprise.
    
- Apple iOS 8.0 ou version ultérieure.
    
    Google Android 4.0 ou version ultérieure (y compris Samsung KNOX Standard 4.0 ou version ultérieure). Pour plus d'informations, voir [Appareils pris en charge par Intune](https://go.microsoft.com/fwlink/p/?linkid=852307).
    
- Si des applications Office sont déjà installées sur les ordinateurs des utilisateurs, lisez [Se préparer à l'installation du client Office](prepare-for-office-client-deployment.md) pour comprendre les mesures que vous devrez peut-être prendre pour pouvoir configurer Microsoft 365 Entreprise en vue d'installer Office 2016 sur les ordinateurs des utilisateurs.
