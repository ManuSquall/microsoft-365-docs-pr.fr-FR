<span data-ttu-id="d477d-p101">[] Ce paramètre est désactivé par défaut. Nous vous recommandons cependant de commencer par vérifier que toutes les conditions préalables sont remplies. Les utilisateurs doivent disposer d'une licence pour Microsoft 365 Entreprise pour que ce paramètre installe le logiciel sur l'appareil en plus des autres exigences décrites dans [Se préparer pour le déploiement du client Office par Microsoft 365 Business](prepare-for-office-client-deployment.md).</span><span class="sxs-lookup"><span data-stu-id="d477d-p101">This setting is off by default. Users have to be licensed for Microsoft 365 Business for this setting to push the software to the device in addition to other requirements described in [Prepare for Office client deployment by Microsoft 365 Business](prepare-for-office-client-deployment.md).</span></span> 
  
<span data-ttu-id="d477d-p102">Nous vous recommandons d'accepter le paramètre par défaut lors de l'installation.
