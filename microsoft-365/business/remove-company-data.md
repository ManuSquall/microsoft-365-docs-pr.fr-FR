# <a name="remove-company-data-from-devices"></a><span data-ttu-id="8ebfd-103">Supprimer des données d'entreprise sur des appareils</span><span class="sxs-lookup"><span data-stu-id="8ebfd-103">Remove company data from devices</span></span>

## <a name="remove-company-data"></a><span data-ttu-id="8ebfd-104">Supprimer les données de l'entreprise</span><span class="sxs-lookup"><span data-stu-id="8ebfd-104">Remove company data</span></span>

<span data-ttu-id="8ebfd-p101">Vous pouvez utiliser Microsoft 365 Entreprise pour supprimer les données d'entreprise protégées par Microsoft 365 présentes sur les [appareils](app-protection-settings-for-android-and-ios.md) ou [PC Windows](protection-settings-for-windows-10-devices.md) de vos utilisateurs. **Si vous supprimez des données d'entreprise d'un appareil, vous ne pourrez pas les restaurer**.</span><span class="sxs-lookup"><span data-stu-id="8ebfd-p101">You can use Microsoft 365 Business to remove company data that your users have on their [devices](app-protection-settings-for-android-and-ios.md) or [Windows PCs](protection-settings-for-windows-10-devices.md) that are protected by Microsoft 365. **If you remove company data from a device, you cannot restore it later**.</span></span> 
  
1. <span data-ttu-id="8ebfd-107">Connectez-vous à [Microsoft 365 Business](https://portal.office.com) avec des informations d'identification d'administrateur général.</span><span class="sxs-lookup"><span data-stu-id="8ebfd-107">Sign in to [Microsoft 365 Business](https://portal.office.com) with global admin credentials.</span></span> 
    
2. <span data-ttu-id="8ebfd-108">Dans la carte **Actions d'appareil** du Centre d'administration, sélectionnez **Supprimer les données de l'entreprise**.</span><span class="sxs-lookup"><span data-stu-id="8ebfd-108">In the admin center, on the **Device actions** card, choose **Remove company data**.</span></span>
    
    ![On the Devices card, choose Remove company data](media/b6fcf74b-0d7d-4e1a-894f-40f9d4a215b8.png)
  
3. <span data-ttu-id="8ebfd-110">Dans le volet **Supprimer les données de l'entreprise**, sélectionnez ou recherchez l'utilisateur dont vous souhaitez supprimer les données, puis sélectionnez son nom.</span><span class="sxs-lookup"><span data-stu-id="8ebfd-110">On the **Remove company data** pane, choose or search for a user who's data you want to remove, and choose the name.</span></span> 
    
4. <span data-ttu-id="8ebfd-111">Dans le volet suivant, dans la liste **Appareils gérés**, sélectionnez les appareils sur lesquels vous souhaitez supprimer les données, puis sélectionnez **Suivant**.</span><span class="sxs-lookup"><span data-stu-id="8ebfd-111">On the next pane, select the device or devices from the **Managed devices** list to remove data from and choose **Next**.</span></span> 
    
    ![On the remove comapany data pane, select the device from which you want to remove the data.](media/f3725ff9-ebdb-4c13-9523-b2df362640cf.png)
  
5. <span data-ttu-id="8ebfd-113">Dans le volet **Voulez-vous vraiment supprimer les données d'entreprise sur les appareils ?**, sélectionnez **Confirmer** \> **Fermer**.</span><span class="sxs-lookup"><span data-stu-id="8ebfd-113">On the **Are you sure you want to remove company data on the devices** pane, choose **Remove** \> **Close**.</span></span>
