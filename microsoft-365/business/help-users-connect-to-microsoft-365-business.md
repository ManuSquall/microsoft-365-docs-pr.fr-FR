# <a name="help-users-connect-to-microsoft-365-business"></a>Aider les utilisateurs à se connecter à Microsoft 365 Entreprise

[] Une fois l'installation terminée, vos stratégies par défaut sont en place et prêtes à être appliquées. Sur les appareils mobiles, les utilisateurs devront télécharger les applications Office. Les stratégies sont appliquées ensuite lorsque l'utilisateur se connecte à l'aide de son compte professionnel ou scolaire. Il reste un peu de travail à faire avant que les stratégies ne s'appliquent aux PC Windows, par exemple pour une mise à niveau sur place. Référencez et partagez les rubriques suivantes pour aider les utilisateurs à finaliser la configuration et la connexion.
  
[Configurer des appareils Windows pour les utilisateurs de Microsoft 365 Business](set-up-windows-devices.md)
  
[Utiliser le guide étape par étape pour ajouter des appareils et un profil Autopilot](add-autopilot-devices-and-profile.md)
  
[Configurer des appareils mobiles pour les utilisateurs Microsoft 365 Entreprise](set-up-mobile-devices.md)
