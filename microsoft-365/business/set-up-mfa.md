# <a name="multi-factor-authentication"></a>	Authentification multifacteur

L’authentification multifacteur (MFA) exige que les utilisateurs se connectent avec une deuxième méthode pour vérifier leur identité avec un appel téléphonique ou avec une application d’authentificateur.

## <a name="set-up-mfa-in-the-microsoft-365-admin-center"></a>Configurer MFA dans le centre d’administration Microsoft 365

[![Étiquette pour vous informer le centre d’administration change et vous trouverez plus de détails sur aka.ms/aboutM365preview.](media/m365admincenterchanging.png)](https://docs.microsoft.com/office365/admin/microsoft-365-admin-center-preview)

1. Connectez-vous au [Centre d’administration Microsoft 365](https://admin.microsoft.com) à l’aide de vos informations d’identification d’administrateur général. 
2. Dans le volet de navigation de gauche, sélectionnez **configuration**.
3. Sur la page installation, choisissez **affichage** sur la carte **Multi-Factor Authentication (MFA)** .
4. Sur la page **activer l’authentification multifacteur** , sélectionnez **prise en main**ou **gérer** si vous avez déjà configuré MFA et souhaitez apporter des modifications. 

  :::image type="content" source="media/turnonmfa.png" alt-text="Capture d’écran de l’activation de la page MFA.":::

5. Dans le volet de **sécurité de connexion Strenghen** , cochez la ou l’une des options d' **authentification multifacteur requises pour les administrateurs**, puis choisissez **créer une stratégie**.
