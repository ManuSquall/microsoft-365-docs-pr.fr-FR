# <a name="add-more-users"></a>Ajouter d’autres utilisateurs

Cet article s’applique à Microsoft 365 Business Premium.

## <a name="add-new-users"></a>Ajouter de nouveaux utilisateurs

Regardez une courte vidéo sur l’ajout d’un utilisateur. <br><br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FOfN] 

Si vous avez trouvé cette vidéo utile, consultez les [séries de formations complètes pour les petites entreprises et les nouveaux utilisateurs de Microsoft 365](https://support.office.com/article/6ab4bbcd-79cf-4000-a0bd-d42ce4d12816).

Pour ajouter un utilisateur :

1. Accédez au Centre d’administration à l’adresse <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>. 
2. Dans le volet de navigation de gauche **, choisissez utilisateurs** \> **actifs**.
3. Sur la page **utilisateurs actifs** , sélectionnez **Ajouter un utilisateur**.
4. Dans le volet **nouvel utilisateur** , entrez les informations requises. 
  
    Vous pouvez entrer davantage d’informations sous **informations de contact**, choisir comment définir le mot de passe sous paramètre **de mot de passe** et attribuer des rôles sous **rôles**.
      
    ![Enter user information in the New user card](../media/f04d39ca-48be-4868-8330-8552a4754c8b.png)
      
    Dans la section licences de produit, définissez le paramètre de licence de produit **Microsoft 365 Business Premium** **sur activé**.
      
    ![Set the license setting to On position](../media/7404f7f7-93bc-44a3-9ffb-4208b5b17402.png)
  
Pour plus d’informations sur l’ajout d’utilisateurs, consultez la rubrique [Ajouter des utilisateurs individuellement ou en bloc](https://docs.microsoft.com/office365/admin/add-users/add-users).
