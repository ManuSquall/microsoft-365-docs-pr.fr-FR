# <a name="microsoft-365-business-security-features"></a>Fonctionnalités de sécurité Microsoft 365 entreprise

Microsoft 365 Business offre des fonctionnalités de sécurité est simplifié pour protéger vos données sur PC, les téléphones et tablettes.
    
## <a name="microsoft-365-business-admin-center-security-features"></a>Fonctionnalités de sécurité de centre d’administration Microsoft 365 entreprise

Vous pouvez gérer de nombreuses fonctionnalités de sécurité Microsoft 365 Business dans le centre d’administration, qui vous donne un moyen simple d’activer ou désactiver les ces fonctionnalités. Dans le centre d’administration, vous pouvez effectuer les éléments suivants :
  
![Screenshot of the Devices card in the admin center](media/9982e784-dbf9-4a76-a159-bb3e2e5aa23f.png)
  
- [Définir les paramètres de gestion des applications pour les appareils Android ou iOS](app-protection-settings-for-android-and-ios.md) . 
    
    Ces paramètres incluent la suppression des fichiers à partir d’un périphérique inactif après une période donnée, chiffrer les fichiers de travail, exiger que les utilisateurs définir un code confidentiel, les etc..
    
- [Définir les paramètres de protection d’application pour les appareils Windows 10](protection-settings-for-windows-10-devices.md) . 
    
    Ces paramètres peuvent être appliqués à des données de société à la société ou périphériques personnels.

    Vous pouvez réinitialiser à distance les données de l’entreprise si un périphérique est perdu, volé, ou un employé quitte votre société.

    Vous pouvez réinitialiser tous les périphériques Windows 10 dont les paramètres de protection de périphériques qui leur sont appliquées.

## <a name="set-up-advanced-threat-protection-features"></a>Configurer les fonctionnalités de protection contre les menaces avancées

- **Protéger contre les pièces jointes non sécurisés :** DAV identifie le contenu malveillant à l’ouverture des pièces jointes dans un environnement virtuel et l’analyse du comportement qui en résulte. Le contenu est évalué pour déterminer son intention (normal ou malveillant), et DAV bloque la livraison des pièces jointes non sécurisés, vous protéger contre les schémas de phishing et infection de logiciels. Pour activer la protection de la pièce jointe, voir [définir des stratégies Office 365 DAV approuvés en pièce jointe](https://support.office.com/article/078eb946-819a-4e13-8673-fe0c0ad3a775).
    
- Protéger votre environnement lorsque les utilisateurs cliquent sur les liens malveillants : DAV examine également les liens dans le message électronique à la fois un utilisateur clique dessus. Si un lien est non sécurisé, l’utilisateur est averti ne pas sur le site ou informé que le site a été bloqué. Cela permet de protéger contre le phishing. Vous pouvez [définir des stratégies Office 365 DAV fiables liens](https://support.office.com/article/bdd5372d-775e-4442-9c1b-609627b94b5d#reveddefaultscc) ou [définir des stratégies Office 365 DAV fiables liens](https://support.office.com/article/bdd5372d-775e-4442-9c1b-609627b94b5d#addemailpolscc).

## <a name="set-up-azure-information-protection-features"></a>Configurer les fonctionnalités de Protection des informations Azure

Protection des informations Azure (AIP) est une solution basée sur le cloud qui aide une entreprise à classer et protéger le cas échéant, les documents et les messages électroniques en appliquant les étiquettes. Étiquettes peuvent être appliquées automatiquement par les administrateurs qui définissent les règles et conditions, manuellement par les utilisateurs, ou une combinaison où les utilisateurs reçoivent des recommandations.

La possibilité d’appliquer les restrictions suivantes lors de l’envoi de messages électroniques dans Outlook sur le site web est automatiquement activée pour tous les utilisateurs :
  
- **Ne pas transférer**: destinataires peuvent lire le message, mais ils ne peuvent pas transférer, imprimer ou copier du contenu
    
- **Chiffrer**: l’intégralité du message est chiffré. Les destinataires doivent suivre des étapes supplémentaires pour vérifier leur identité avant d’accéder au contenu chiffré et ne peut pas supprimer le chiffrement.
    
- **Confidentiel**: donne les employés de votre organisation à toutes les autorisations pour le contenu de messagerie et des pièces jointes, mais pas pour les personnes extérieures à votre organisation. Les propriétaires de données peuvent effectuer le suivi et révoquer le contenu à tout moment.
    
- **Hautement confidentielles**: cette restriction peut être appliquée à des données hautement confidentielles, ce qui permet aux employés d’afficher, modifier et répondre, mais pas transférer, imprimer ou copier les données. Les propriétaires de données peuvent effectuer le suivi et révoquer le contenu à tout moment.

### <a name="make-sure-azure-information-protection-is-activated"></a>Assurez-vous que la Protection des informations Azure est activée.

Pour vérifier que le point d’installation administrative est activée :

1. Connectez-vous au [portail Azure](https://portal.azure.com/).

2. Sélectionnez **tous les services** et le type de *Protection des informations Azure* dans la **Zone de recherche**.

3. Une fois que les résultats s’affichent, cliquez sur Démarrer suivante à **La Protection des informations Azure** à ajouter aux Favoris et facile à trouver ultérieurement.

4. Activez **La Protection des informations Azure** \> **activation de la Protection** et assurez-vous que le statut est défini à activée.
