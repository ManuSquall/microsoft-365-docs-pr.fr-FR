# <a name="configure-windows-10-devices"></a>Configurer les appareils Windows 10

Lors de l’installation, vous créez une stratégie d’appareil pour Windows 10 qui s’applique à tous les utilisateurs de votre organisation. Lorsqu’un utilisateur se connecte à une application Office sur son appareil Windows 10 avec son compte professionnel ou scolaire, les paramètres que vous configurez sont appliqués automatiquement afin de s’assurer que l’utilisateur se connecte de manière sécurisée.
  
Nous vous recommandons d'accepter la configuration par défaut lors de l'installation. Après l’installation, vous pouvez créer d’autres stratégies et les cibler pour des groupes d’utilisateurs spécifiques.
  
Pour ajouter une stratégie pour gérer les configurations d'appareils Windows 10, consultez [Définir des configurations d'application pour les PC Windows 10](protection-settings-for-windows-10-pcs.md).
