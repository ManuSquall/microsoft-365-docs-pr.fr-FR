# <a name="get-access-to-microsoft-bookings"></a>Accès à Microsoft Bookings

Cette rubrique décrit comment accéder aux réservations pour la première fois. Il vous indique également comment activer ou désactiver les réservations dans votre organisation. Il explique également comment attribuer des licences à vos utilisateurs afin qu’ils puissent accéder et utiliser les réservations.

## <a name="access-bookings-for-the-first-time"></a>Pour la première fois

Si votre administrateur informatique vous a accordé l’accès aux réservations, vous pouvez accéder à l’application via Office Online.

La première fois que vous utilisez des [réservations](https://outlook.office.com/bookings/onboarding) , vous créez un calendrier et configurez votre entreprise.

1. Sélectionnez **obtenir maintenant** et sélectionnez, **Ajouter un calendrier de réservation**. Si cette page n’est pas la première fois, vous pouvez sélectionner un autre calendrier de réservation ou effectuer une recherche pour un autre calendrier de réservation.

:::image type="content" source="media/bookings - get it now.png" alt-text="Capture d’écran : page des réservations sur Get It Now. Sélectionnez obtenir maintenant pour commencer à utiliser les réservations":::

2. Entrez le nom de votre entreprise et le type d’entreprise, puis cliquez sur **Continuer**.

Vous êtes maintenant prêt à configurer les réservations de votre organisation. Vous pouvez accéder à la page réservations avec ce [lien](https://outlook.office.com/bookings/onboarding), à partir du lanceur d’applications ou de Office.com.
