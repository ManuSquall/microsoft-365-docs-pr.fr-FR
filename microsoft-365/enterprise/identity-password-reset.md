# <a name="step-4-simplify-password-resets"></a>Étape 4 : Simplifier les réinitialisations du mot de passe

*Cette étape est facultative et s’applique aux versions E3 et E5 de Microsoft 365 Entreprise*

![](./media/deploy-foundation-infrastructure/identity_icon-small.png)

Lors de cette étape, vous allez activer la réinitialisation du mot de passe libre-service (SSPR) pour permettre aux utilisateurs de réinitialiser ou de déverrouiller leurs mots de passe ou comptes. Pour être averti de tout abus ou utilisation incorrecte, vous pouvez utiliser le compte-rendu détaillé qui indique lorsque des utilisateurs accèdent au système, de même que les notifications.

Reportez-vous aux [instructions pour activer la réinitialisation de mot de passe](https://docs.microsoft.com/azure/active-directory/active-directory-passwords-best-practices).
