<span data-ttu-id="f4e69-p105">Si vous décidez de capturer l’image, il est préférable d’automatiser le processus autant que possible afin de garantir la meilleure qualité d’image et d’obtenir un processus renouvelable. By starting with a thin base image, you can layer on required apps, languages and configurations dynamically tailored to users.</span></span>

<span data-ttu-id="f4e69-122">Lors du processus de création et de capture, les outils tels que Microsoft Endpoint Configuration Manager (Current Branch) et Microsoft Deployment Toolkit utilisent l’outil de préparation du système (ou Sysprep), ainsi que la commande « Généraliser » pour resceller votre image avant de capturer l’installation de Windows 10 sous forme d’image.</span><span class="sxs-lookup"><span data-stu-id="f4e69-122">During the build and capture process, tools like Microsoft Endpoint Configuration Manager (Current Branch) and the Microsoft Deployment Toolkit use the System Preparation Tool – or Sysprep – along with the “Generalize” command to reseal your image before they capture the Windows 10 installation as an image.</span></span>
