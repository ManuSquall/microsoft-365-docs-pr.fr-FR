# <a name="step-1-prepare-your-network-for-microsoft-365"></a>Étape 1 : Préparer le réseau de votre organisation pour Microsoft 365

*Cette étape est requise et s’applique aux versions E3 et E5 de Microsoft 365 Entreprise*

![](./media/deploy-foundation-infrastructure/networking_icon-small.png)

Dans l’Étape 1, vous devez :

- évaluer et ajuster la bande passante réseau pour les liens internes et les connexions Internet afin de prendre en compte le trafic vers les services cloud de Microsoft 365 Entreprise ;
- aligner votre réseau avec une [architecture de référence Office 365](https://docs.microsoft.com/office365/enterprise/office-365-network-connectivity-principles#BKMK_P2) ;
- planifier les modifications, les piloter et tester ensuite l’adéquation de vos besoins de latence de trafic et de bande passante.

Comme point de contrôle intermédiaire, consultez les [critères de sortie](networking-exit-criteria.md#crit-networking-step1) correspondant à cette étape.
