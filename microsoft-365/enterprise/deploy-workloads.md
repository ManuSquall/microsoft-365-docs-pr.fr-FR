# <a name="microsoft-365-enterprise-workloads-and-scenarios"></a>Scénarios et charges de travail Microsoft 365 Entreprise

Pour profiter des avantages de la créativité et du travail d’équipe de Microsoft 365 Entreprise, déployez les charges de travail et scénarios suivants afin d’agir au niveau le plus élevé de votre infrastructure de base :

- [Microsoft Teams](teams-workload.md)
- [Exchange Online](exchangeonline-workload.md)
- [SharePoint Online](sharepoint-online-onedrive-workload.md)
- [Migration vers Microsoft 365 Entreprise](migration-microsoft-365-enterprise-workload.md)
- [Sites SharePoint Online et Microsoft Teams pour les données hautement réglementées](teams-sharepoint-online-sites-highly-regulated-data.md)

La figure suivante illustre les charges de travail dans le contenu de déploiement de Microsoft 365 Entreprise global.

![](./media/deploy-workloads/m365-deploy-content-arch-workloads.png)

Vous devez déployer les charges de travail une fois que vous avez parcouru toutes les phases de l’[infrastructure de base](deploy-foundation-infrastructure.md)
