<span data-ttu-id="f6182-109">L’authentification moderne est activée par le biais de l’utilisation d’une bibliothèque d’authentification, telle que la bibliothèque d’authentification Active Directory (ADAL) ou la bibliothèque d’authentification Microsoft (MSAL).</span><span class="sxs-lookup"><span data-stu-id="f6182-109">Modern authentication is enabled through the use of an authentication authenticate and authorize access to Microsoft 365 resources.</span></span> <span data-ttu-id="f6182-111">L’authentification moderne s’appuie sur OAuth et fournit un mécanisme sécurisé permettant aux clients d’accéder aux services Microsoft 365, sans avoir à accéder aux informations d’identification de l’utilisateur.</span><span class="sxs-lookup"><span data-stu-id="f6182-111">Modern
