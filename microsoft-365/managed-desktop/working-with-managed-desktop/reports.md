# <a name="work-with-reports"></a>Utiliser les rapports

Microsoft Managed Desktop fournit des rapports dont les administrateurs informatiques de votre client peuvent se servir pour comprendre les différents aspects de la population des appareils. Vous accédez aux rapports dans le [portail d’administration ](https://aka.ms/mmdportal)sous **Reporting**.



## <a name="device-inventory-report"></a>Rapport d’inventaire des appareils

La vue **périphériques** propose un onglet **Exporter tout** qui [télécharge un rapport d’inventaire détaillé](device-inventory-report.md).
