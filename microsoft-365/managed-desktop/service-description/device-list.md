# <a name="microsoft-managed-desktop-devices"></a>Appareils de bureau gérés Microsoft 

Microsoft Managed Desktop évalue régulièrement les appareils à inclure dans le service. Cette rubrique décrit les conditions requises pour l’enregistrement d’un appareil dans le bureau géré Microsoft.

Pour plus d’informations sur le bureau géré Microsoft, consultez [la documentation et les ressources du bureau géré Microsoft](https://docs.microsoft.com/microsoft-365/managed-desktop/).

    - Dispose de la version la plus récente de Microsoft Office 365 cliquez pour exécuter ou publier immédiatement la version précédente;

- N’ont pas terminé l’expérience de première exécution de Windows.
