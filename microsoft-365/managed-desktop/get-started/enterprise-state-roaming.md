# <a name="enable-enterprise-state-roaming"></a>Activer l’itinérance du statut Entreprise

Pour une expérience optimale avec Microsoft Managed Desktop, activez l' [itinérance d’état d’entreprise](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-overview), qui permet aux utilisateurs de synchroniser les données des paramètres d’application et d’utilisateur dans le Cloud de manière sécurisée. Cela signifie qu’ils auront la même expérience, quel que soit l’appareil Windows auquel ils se connectent. Par exemple, si vous remplacez un de ses périphériques de bureau géré Microsoft par un nouveau, il s’affiche et se comporte exactement comme le dernier.

Pour activer l’itinérance de l’état d’entreprise, suivez les étapes de la procédure [activer l’itinérance d’état d’entreprise dans Azure Active Directory](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-enable), puis revenez à cette documentation.

Si vous rencontrez des difficultés avec l’itinérance de l’état d’entreprise, contactez le [support](../working-with-managed-desktop/admin-support.md)d’administration.
