# <a name="enable-enterprise-state-roaming"></a>Activer Enterprise State Roaming

Pour une expérience de bureau géré Microsoft, activez Enterprise [State Roaming,](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-overview)qui permet aux utilisateurs de synchroniser en toute sécurité les données des paramètres utilisateur et d’application sur le cloud. Cela signifie qu’ils auront la même expérience, quel que soit l’appareil Windows auquel ils se connectent. Par exemple, si vous remplacez l’un de ses appareils Bureau géré Microsoft par un nouvel appareil, il se comportera exactement comme le dernier.

Pour activer Enterprise State Roaming, suivez les étapes de l’étape Activer l’itinérance d’état d’entreprise dans [Azure Active Directory,](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-enable)puis revenir à cette documentation.

Si vous avez des difficultés avec l’itinérance de l’état d’entreprise, contactez le [support de l’administrateur.](../working-with-managed-desktop/admin-support.md)
