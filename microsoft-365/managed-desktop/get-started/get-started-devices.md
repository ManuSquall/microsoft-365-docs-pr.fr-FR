# <a name="get-your-users-ready-to-use-devices"></a>Préparer vos utilisateurs à l’utilisation des appareils

Une fois qu’un appareil bureau géré Microsoft est entre les mains de votre utilisateur, sa mise en place est rapide et facile. Les appareils sont pré-configurés avec la version actuelle de Windows, et les configurations et les applications sont installées à partir du cloud à mesure que l’utilisateur termine l’installation. 
 
Pour faciliter encore cette tâche, nous proposons un guide qui guide vos utilisateurs tout au long de la configuration initiale et fournit des ressources d’aide à la fois pour l’installation et pour une utilisation ultérieure, si nécessaire. Vous pouvez personnaliser ce guide pour inclure certains détails spécifiques à votre organisation. Vous distribuez ensuite le guide directement à vos utilisateurs avec leur appareil. Il existe deux versions, une pour les organisations qui utilisent le système de support de Microsoft (l’application Obtenir de l’aide), une pour celles qui fournissent une assistance alternative pour leurs utilisateurs.

## <a name="to-prepare-the-guide"></a>Pour préparer le guide

1. Téléchargez la version appropriée à votre organisation :
- [Bureau géré Microsoft : prise en](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-custom-v3.pdf) charge de votre appareil (obtenir une version d’aide) : utilisez cette version si vos utilisateurs utiliseront le système de support de Microsoft.
- [Bureau géré Microsoft - Prise en charge de votre appareil](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-custom-v2.pdf) : utilisez cette version si vos utilisateurs n’utiliseront pas le système de support de Microsoft. 
2. Utilisez n’importe quelle application capable d’ouvrir des fichiers PDF pour fournir des détails pertinents pour votre organisation :
    - Nom du réseau à qui vos utilisateurs doivent se connecter pour poursuivre l’installation (étape 3 du guide)
    - Nom du compte client Azure de votre organisation (étape 4 du guide)
    - Coordonnées du support technique interne de votre organisation (en haut de la deuxième page)
3. Enregistrez le fichier PDF modifié, puis distribuez-le à vos utilisateurs. 

## <a name="ready-to-use-guide"></a>« Guide prêt à l’emploi »

Nous fournissons également une version plus générique du guide pour les organisations qui n’ont pas besoin de la personnaliser. Il existe deux versions, une pour les organisations qui utilisent le système de support de Microsoft (l’application Obtenir de l’aide), une pour celles qui fournissent une assistance alternative pour leurs utilisateurs. 

- [Bureau géré Microsoft : prise](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-v3.pdf) en charge de votre appareil (obtenir la version d’aide prête à l’emploi) : utilisez cette version si vos utilisateurs utiliseront le système de support de Microsoft.
- Bureau géré Microsoft : prise en charge de votre appareil  [(prêt à l’emploi)](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-v2.pdf) : utilisez cette version si vos utilisateurs n’utiliseront pas le système de support de Microsoft.
